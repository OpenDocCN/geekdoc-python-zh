## 下载

### Docker

```
docker pull apachecn0/pythonland-zh
docker run -tid -p <port>:80 apachecn0/pythonland-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install pythonland-zh
pythonland-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pythonland-zh
pythonland-zh <port>
# 访问 http://localhost:{port} 查看文档
```
