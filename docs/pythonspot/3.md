## 运行 Python 代码

python 程序应保存为扩展名为`.py`的文件。试试这个代码：

```py
print("Hello World!")
print("This is a Python program.")

```

预期产出：

```py
Hello World!
This is a Python program

```

如果使用解释器，请使用：

```py
python program.py

```
