### 1.1 历史

## 1.7 Python 注释

可以使用注释来解释 Python 代码。使用注释可以使代码更具可读性。在测试代码时，可以使用注释来停止执行。注释以 # 开头，Python 会忽略它们：

### 单行注释

```
#This is a comment
print ("Hello, World!")
```

Python 允许在行尾添加注释，忽略前面的文本。

```
print ("Hello, World!") #This is a comment
```

也可以使用注释来阻止 Python 执行代码，以及为代码提供解释：

```
#cheering for friend
print ("Cheers! friend")
```

### 输出屏幕：

![](img/c5c3113c0233e765f6875c52296f229f_12_0.png)

### 多行注释

实际上，Python 中没有多行注释的语法。你可以使用 # 符号来表示多行注释。

### 示例：

```
#commenthere
#writtenin
#multipleline
print("Hello, World!")
```

Python 不会使用未赋值给变量的字符串字面量，因此你可以在代码中添加一个多行字符串（使用三引号），并在其中编写注释：

示例：

```
"""
multi
line
comment
"""
print("Hey, there!")
```

## 1.9 Python 中的标识符

Python 标识符是用于标识变量、函数、类、模块或其他对象的名称。标识符以字母 A 到 Z 或 a 到 z 或下划线 (_) 开头，后跟零个或多个字母、下划线和数字 (0 到 9)。

- Python 不允许在标识符中使用标点字符，如 @、$ 和 %。
- Python 是一种区分大小写的编程语言。
- 因此，Manpower 和 manpower 在 Python 中是两个不同的标识符。

**声明 Python 标识符的规则？**

- Python 允许在标识符名称中使用下划线 (_)、小写字母 (a-z)、大写字母 (A-Z) 和数字 (0-9)。
- 名称必须以非数字字符开头。
- 标识符名称中不能只包含数字。
- 与大多数其他语言一样，Python 中的标识符名称区分大小写。（'Ash' 和 'ASH' 不是同一回事。）
- 如果用户以开头下划线命名标识符，不会显示错误。
- 尽管 PEP-8 标准规则建议将行长度保持在 79 个字符以内，但标识符名称可以是任意长度。
- 关键字和标识符的名称不应重合。（用户可以通过输入 `help()` 后跟 "keywords" 来查看 Python 中所有关键字的列表，以避免问题。）
- 不需要前导双下划线 ($)。

以下是正确命名标识符的示例：

```
>>> My_language = "Python"
>>> myName = "xyz"
>>> __username__ = "abc12"
```

以下是错误命名标识符的示例：

```
>>> 7hello = "hello"
>>> @hello = "hello"
>>> __user@7= "abc12"
```

## 1.12 Python 中的变量

什么是变量？

变量只是用于存储值的指定内存空间。这意味着当你创建一个变量时，你会预留一些内存。解释器根据变量的数据类型分配内存并确定可以放入预留内存的内容。

因此，你可以通过为变量指定不同的数据类型来存储整数、小数或字符。Python 变量不需要显式声明来预留内存。当你为变量提供一个值时，声明会自动发生。为了给变量赋值，请使用等号（=）。

### 1.12.1 声明变量的规则

- 变量名必须以字母或下划线字符开头，哪个先出现都可以。
- 变量名不能以数字开头。
- 变量名中只允许使用下划线（_）、数字 0 到 9 以及字母 A 到 Z。
- 变量名区分大小写

变量名是 = 运算符左侧的操作数，其值是右侧的操作数。例如，

```
>>>a= 23
>>>print(a)
**输出：**
>>> a=23
>>> print(a)
23
```

Python 允许你同时将相同的值赋给多个变量。例如，

```
>>>a= 23
>>>a= 25
>>>print(a)
**输出：**
>>> a=23
>>> a=25
>>> print(a)
25
```

### 1.13.2 Python 字符串

Python 将字符串定义为用引号括起来的连续字符集合。Python 接受成对的双引号或单引号。使用切片运算符（`[]` 和 `[:]`）可以获取字符串的子集，索引从字符串开头的 0 开始，到结尾的 -1。

```
mystr = "hello"

```

星号（*）表示重复运算符，加号（+）表示字符串连接运算符。

### 1.13.3 Python 列表

列表是 Python 中最灵活的复合数据类型。列表中的项目用逗号分隔，并用方括号（`[]`）括起来。在某些方面，列表与 C 的数组相似。列表可以包含不同数据类型的元素，这是它与其他集合的一个区别。

```
mylist = ["hello",33,"89"]
```

使用切片运算符（`[]` 和 `[:]`）可以访问列表中存储的值，索引从列表开头的 0 开始，一直到列表末尾的 -1。星号（*）表示重复运算符，加号（+）是列表连接运算符。

```
mylist [0:] 我们将在后续章节中看到更多这些内容。
```

## 3.4 局部作用域和全局作用域

变量的作用域是指程序中使用该变量的区域。当在函数内部定义时，参数和变量对外部是隐藏的。因此，它们的作用域是局部的。

变量存储在内存中的时间被称为其生命周期。函数内部的变量仅在该函数执行期间有效。

当我们从函数返回时，它们就被销毁了。因此，函数不会保留之前调用中变量的值。

下面是一个函数内变量作用域的示例。

```
>>>def my_func():
>>>    x = 10
>>>    print("Value inside function:",x)
>>>x = 20
>>>my_func()
>>>print("Value outside function:",x)
```

输出：

```
E:\python>practice-1.py
Value inside function: 10
Value outside function: 20
```

如图所示，`x`的初始值为20。即使`x`被函数`my_func()`从0更改为10，这对函数外部的值也没有影响。

这是因为函数内部的变量`x`可以与外部变量（对函数而言是全局的）区分开来。它们是两个具有不同作用域的不同变量，尽管共享相同的名称。

然而，函数内部的变量从外部是可见的。它们涵盖了整个世界。

从函数内部，我们可以读取这些值，但无法修改（写入）它们。如果要在函数外部更改变量的值，必须使用`global`关键字将变量声明为全局变量。

编程经常涉及处理文件。Python 有内置的方法来创建、打开和关闭文件，这使得处理文件更加容易。

### 8.1 文件类型

Python 的内置例程支持文件的创建、读取和写入。Python 可以处理两种类型的文件：二进制文件和常规文本文件（以二进制语言、0 和 1 编写）。

1.  **文本文件：** 这种类型的文件中的每一行文本都以一个称为 EOL（行结束符）的唯一字符结尾，在 Python 中它等同于换行符（`'\n'`）。
2.  **二进制文件：** 数据在转换为机器可理解的语言后存储在二进制文件中；这些文件中没有行终止符。

### 8.2 用于访问文件的方法

访问模式控制可以对打开的文件执行的操作类型。当文件被打开时，它描述了文件将如何被使用。此外，这些模式指定了文件句柄在文件中的位置。文件句柄充当光标，指示必须从文件中读取数据或向文件中写入数据的位置。Python 有六种不同的访问模式。

1.  **只读（`'r'`）：** 读取文本文件。句柄放置在文件的开头。如果文件不存在，则引发 I/O 错误。此外，这是默认打开文件的方式。
2.  **打开文件以允许读写（`r+`）。** 句柄位于文件的开头。
3.  **打开文件以进行只写（`'w'`）。** 对于已存在的文件，数据会被覆盖和截断。句柄位于文件的开头。如果文件不存在，则创建它。
4.  **使用 `w+` 命令打开文件以进行读写。** 对于已存在的文件，数据会被覆盖和截断。句柄位于文件的开头。
5.  **只追加（`'a'`）：** 开始向文件中写入。如果文件不存在，则创建一个。句柄位于文件的末尾。新写入的数据将添加在末尾，位于先前写入的数据之后。
6.  **追加和读取（`'a+'`）：** 允许在文件中读写。如果文件不存在，则创建一个。句柄位于文件的末尾。新写入的数据将添加在末尾，位于先前写入的数据之后。

### 8.3 将文件加载到内存中

计算机有两种不同类型的内存：主存储器和辅助存储器。你或任何其他人存储的所有文件都保存在辅助存储器中，因为计算机会在关闭时删除主存储器中的任何数据。因此，在编辑文本文件或仅仅在 Python 中使用它之前，你必须将其加载到主存储器中。Python 使用“文件句柄”与加载到主存储器中的文件进行通信（这就是你的操作系统通过在其内存中搜索文件来给予 Python 访问你打开的文件的权限的方式，如果找到则返回一个文件句柄，然后你就可以处理该文件了）。

## 8.8 Python 中的异常

当程序中出现问题时，Python 的众多内置异常会被引发（程序中的某些内容出错）。Python 解释器会停止运行过程，并将控制权转移给调用过程，直到这些异常得到处理。如果情况未得到处理，程序将会崩溃。

考虑一个软件，其中函数 A 调用函数 B，函数 B 调用函数 C。如果在函数 C 中发生异常但未在该处处理，则异常将传递给 B，然后传递给 A。如果错误从未得到解决，则会显示通知，我们的软件会突然且不可预测地终止。

## 8.9 Python 中捕获异常

Python 中的 `try` 语句可用于管理异常。

`try` 子句包含可能导致异常的关键操作。`except` 子句包含管理异常的代码。

因此，一旦我们确定了异常，就可以决定采取什么行动。这里有一个简单的说明。

```python
>>> # 导入模块 sys 以获取异常类型
>>> import sys
>>> randomList = ['a', 0, 2]
>>> for entry in randomList:
...     try:
...         print("The entry is", entry)
...         r = 1/int(entry)
...         break
...     except:
...         print("Oops!", sys.exc_info()[0], "occurred.")
...         print("Next entry.")
...         print()
>>> print("The reciprocal of", entry, "is", r)
```

在此应用程序中，迭代 `randomList` 列表的值。如前所述，`try` 块包含可能引发异常的代码。

如果没有异常，则跳过 `except` 块，正常操作继续（对于最后一个值）。但如果发生异常，则 `except` 块会处理它（对于第一个和第二个值）。

我们在这里使用 `sys` 模块中的 `exc_info()` 方法打印异常的名称。我们可以看到 `0` 导致 `ZeroDivisionError`，而 `a` 导致 `ValueError`。

## 8.10 Python：捕获特定异常

上面示例中的 `except` 子句没有包含任何特定的异常。这将捕获所有异常并统一处理每种情况，因此这不是一个好的编程方法。可以指定 `except` 子句应涵盖的异常。

如果发生异常，附加到 `try` 子句的任何数量的 `except` 子句中只有一个会被执行。

为了在 `except` 子句中指示多个异常，我们可以使用值的元组。这里演示了伪代码。

```python
>>> try:
...     # 执行某些操作
...     pass
>>> except ValueError:
...     # 处理 ValueError 异常
...     pass
>>> except (TypeError, ZeroDivisionError):
...     # 处理多个异常
...     # TypeError 和 ZeroDivisionError
...     pass
>>> except:
...     # 处理所有其他异常
...     pass
```

## 8.11 Python 中的 Try 与 else

如果 `try` 内的代码块在没有任何问题的情况下执行，你可能偶尔希望运行另一段代码。在某些情况下，你可以将 `try` 语句与可选的 `else` 关键字配对。

示例：

```python
...     print("Not an even number!")

...     reciprocal = 1/num

```

## 8.12 Python 中的 Try 与 Finally

在 Python 中，`finally` 子句是 `try` 语句的可选补充。这个子句通常用于释放外部资源，并且总是被执行。

例如，我们可能在通过网络连接到远程数据中心时使用文件或图形用户界面（GUI）。

在任何这些场景中，无论程序是否成功运行，我们都必须在终止前清除资源。`finally` 子句执行这些操作（关闭文件、关闭 GUI 或断开网络连接）以确保执行。

这里是一个文件操作示例来演示这一点。

```python
>>> try:
...     f = open("test.txt", encoding='utf-8')
...     # 执行文件操作
>>> finally:
...     f.close()
```

## 9.1 类和对象

Python 是一种面向对象的编程语言。面向对象编程比面向过程编程更强调对象，而后者更注重函数。

对象是数据（变量）和操作这些变量的方法（函数）的集合。类是对象的蓝图，方式类似。

类可以比作房屋的粗略草图或原型。它包含有关楼层、门窗等的所有信息。我们根据这些描述建造房屋。对象就是房屋。

我们可以从一个类生成多个对象，就像使用一个房屋蓝图可以建造多个房屋一样。生成对象的过程称为实例化，对象也称为类的实例。

## 9.2 定义类

Python中的类定义以`class`关键字开头，就像函数定义以`def`关键字开头一样。

类的简要描述可以放在类内部的第一个字符串中，这被称为文档字符串。虽然这不是强制要求的，但强烈建议这样做。

以下是对类的简要解释。

```python
>>>class MyNewClass:
    >>> """This is a docstring. I have created a new class"""
    >>>pass
```

类的每个属性都在一个新的局部命名空间中声明。函数或数据都可以是属性。

它还具有以双下划线开头的独特属性，例如。例如，该类的文档字符串由`__doc__`提供。

一旦定义了一个类，就会创建一个同名的新类对象。这个类对象使我们能够访问各种特性，并允许我们创建属于该类的全新对象。

示例：

```python
>>>class Person:
    >>> """This is a person class"""
    >>>age = 10
    >>>def greet(self):
    >>>    print('Hello')
>>>print(Person.age)
>>>print(Person.greet)
>>>print(Person.__doc__)
```

## 9.3 创建对象

我们观察到类对象可以访问许多特性。

此外，它还可以用来实例化该类对象类型的新实例。函数调用类似于创建对象的过程。

```python
>>> harry = Person()
```

这将创建一个名为Harry的新对象实例。通过使用对象名称的前缀，我们可以访问对象的属性。

属性包括数据和方法。对象的方法是该类对应的函数。

换句话说，由于`Person.greet`是一个函数对象（一个类属性），`Person.greet`将是一个方法对象。

示例：

```python
>>>class Person:
>>>    "This is a person class"
>>>    age = 10
>>>    def greet(self):
>>>        print('Hello')

# create a new object of Person class
>>>harry = Person()
>>>print(Person.greet)
>>>print(harry.greet)

# Calling object's greet() method
>>>harry.greet()
```

你可能注意到了类内部函数定义中的`self`参数，但我们刚刚调用了Harry的方法`greet()`，没有传递任何参数。它仍然执行了。

这是因为当对象调用其方法时，对象本身总是作为第一个参数传递。因此，`Person.greet`等同于`harry.greet()`。

通常，使用n个参数列表调用一个方法，等同于使用一个参数列表调用相应的函数，该参数列表是通过将方法的对象放在第一个参数之前构成的。

这些解释决定了对象本身必须是类函数的第一个参数。通常用`self`来描述这个参数。虽然可以给它起不同的名字，但我们强烈建议这样做。

## 9.4 Python中的构造函数

特定函数是具有特殊意义的类函数，并以双下划线开头。

`__init__()`方法非常重要。每次创建该类的新对象时，都会调用这个特定的函数。

在面向对象编程中，这种函数也被称为构造函数（OOP）。通常，我们用它来初始化每个变量。

示例：

```python
>>>class ComplexNumber:
>>>    def __init__(self, r=0, i=0):
>>>        self.real = r
>>>        self.imag = i
>>>    def get_data(self):
>>>        print(f'{self.real}+{self.imag}j')
>>># Create a new ComplexNumber object
>>>num1 = ComplexNumber(2, 3)
>>># Call get_data() method
>>>num1.get_data()
>>># Create another ComplexNumber object
>>># and create a new attribute 'attr'
>>>num2 = ComplexNumber(5)
>>>num2.attr = 10
>>>print((num2.real, num2.imag, num2.attr))
```

## 9.8 继承

得益于继承，我们可以定义一个包含另一个类所有方法和属性的类。被继承的类称为父类或基类。从另一个类继承的类称为“子类”，有时也称为“派生类”。

## 创建父类

语法与创建任何其他类相同，因为任何类都可以成为父类：

示例

```python
class Person:

        self.lastname = lname

    def printname(self):
        print(self.firstname, self.lastname)

# printname 方法：

```

## 创建子类

在创建子类时，将父类作为参数添加，你就可以设计一个继承其功能的类：

我们目前开发的子类继承了父类的属性和函数。

我们希望给子类添加 `__init__()` 函数（而不是 `pass` 关键字）。

- 每次使用该类创建新对象时，`__init__()` 函数都会被自动调用。
- 你添加的 `__init__()` 函数将阻止子类继续使用父类的 `__init__()` 函数。
- 子类的 `__init__()` 函数会覆盖从父类继承的任何内容。

示例：

```python

        self.first_name = name1
        self.second_name = name2
        self.third_name = name3

        print(self.first_name, self.second_name, self.third_name)

class Student(Person):
    pass

x = Student("kaushik", "akash", "varun")
x.printname()
```

## 继承的类型

继承的类型取决于涉及的子类和父类的数量。

Python 中有四种继承类型：

1.  **单继承**：单继承允许派生类从单个父类继承属性，从而可以重用现有代码并添加新功能。

![](img/c5c3113c0233e765f6875c52296f229f_109_0.png)

示例：

```python
class Parent:
    def func1(self):
        print("This function is in parent class.")

class Child(Parent):
    def func2(self):
        print("This function is in child class.")

object = Child()
object.func1()
object.func2()
```

2.  **多重继承**：多重继承指的是一种继承类型，其中一个类可以源自多个基类。在多重继承中，所有基类的特性都会传递给派生类。

![](img/c5c3113c0233e765f6875c52296f229f_110_0.png)

示例：

```python
class Mother:
    mothername = ""
    def mother(self):
        print(self.mothername)

class Father:
    fathername = ""
    def father(self):
        print(self.fathername)

class Son(Mother, Father):
    def parents(self):
        print("Father :", self.fathername)
        print("Mother :", self.mothername)

s1 = Son()
s1.fathername = "RAM"
s1.mothername = "SITA"
s1.parents()
```

3.  **多级继承**：在多级继承中，基类和派生类的特性进一步被新的派生类继承。这类似于祖父与孙辈之间的关系。

![](img/c5c3113c0233e765f6875c52296f229f_111_0.png)

示例：

```python
class Grandfather:
    def __init__(self, grandfathername):
        self.grandfathername = grandfathername

class Father(Grandfather):
    def __init__(self, fathername, grandfathername):
        self.fathername = fathername
        Grandfather.__init__(self, grandfathername)

class Son(Father):
    def __init__(self, sonname, fathername, grandfathername):
        self.sonname = sonname
        Father.__init__(self, fathername, grandfathername)
    def print_name(self):
        print('Grandfather name :', self.grandfathername)
        print("Father name :", self.fathername)
        print("Son name :", self.sonname)

s1 = Son('Prince', 'Rampal', 'Lal mani')
print(s1.grandfathername)
s1.print_name()
```

4.  **层次继承**：层次继承是从单个基类创建多个派生类的过程。此软件中存在两个子（派生）类和一个父（基）类。

![](img/c5c3113c0233e765f6875c52296f229f_112_0.png)

示例：

```python
class Parent:
    def func1(self):
        print("This function is in parent class.")

class Child1(Parent):
    def func2(self):
        print("This function is in child 1.")

class Child2(Parent):
    def func3(self):
        print("This function is in child 2")

object1 = Child1()
object2 = Child2()
object1.func1()
object1.func2()
object2.func1()
object2.func3()
```

5.  **混合继承**：混合继承是由多种继承类型组合而成的继承。

![](img/c5c3113c0233e765f6875c52296f229f_113_0.png)

示例：

```python
class School:
    def func1(self):
        print("This function is in school.")

class Student1(School):
    def func2(self):
        print("This function is in student 1. ")

class Student2(School):
    def func3(self):
        print("This function is in student 2.")

class Student3(Student1, School):
    def func4(self):
        print("This function is in student 3.")

object = Student3()
object.func1()
object.func2()
```

## 随机游走

### 11.1 什么是随机游走？

简而言之，随机游走是相对于当前状态以“随机”方式进行后续步骤的过程。这个基本描述可以根据附加条件进行修改，以生成适用于你特定用例的随机游走。现实世界中一些更著名的随机游走包括股票行情走势、基质中活细胞的运动以及粒子的布朗运动。

在这里，我们模拟一维（一维）、二维（二维）和三维（三维）的紧凑随机游走。停止位置用 o 表示，起始点用 + 表示。

这些要求可能因应用而异；例如，游走可能从特定的股票价格开始，或者从显微镜确定的细胞位置开始。步骤选择通常是概率性的，并基于来自先前数据、投影假设、正在测试的假设等的附加信息。

![](img/c5c3113c0233e765f6875c52296f229f_139_0.png)

### 11.2 一维随机游走

一维随机游走是最简单的随机游走类型。在这种类型的随机游走中，移动物体只能在一个维度上移动。在这个例子中，是上下移动。

计划是让物体随机向上或向下移动 n 次，同时将位置数据存储在列表中。

由于物体的运动是一维的，路径在图中由一定高度的线表示。由于很难确定物体在每个阶段的位置，这没有用。为了使图表更具说明性，我们将为过程中的每一步添加一个时间戳。这将其转换为一个二维图表，日期作为 x 轴，物体的位置作为 y 轴。

示例：

```python
import numpy as np
import matplotlib.pyplot as plt
import random

def randomwalk1D(n):
    x, y = 0, 0
    # 生成时间点 [1, 2, 3, ... , n]
    timepoints = np.arange(n + 1)
    positions = [y]
    directions = ["UP", "DOWN"]
    for i in range(1, n + 1):
        # 随机选择向上或向下
        step = random.choice(directions)
        # 将物体向上或向下移动
        if step == "UP":
            y += 1
        elif step == "DOWN":
            y -= 1
        # 记录位置
        positions.append(y)
    return timepoints, positions

time_data, pos_data = randomwalk1D(1000)
plt.plot(time_data, pos_data, 'r-')
plt.title("Python 中的一维随机游走")
plt.show()
```

输出：

### 11.3 二维随机游走

二维随机游走的原理与一维完全相同。物体现在除了上下移动外，还可以向其他方向移动。它也可以向左或向右移动。

在二维场景中，你不再绘制时间。相反，通过将 x、y 坐标对添加到图中，可以观察到游走。这显示了物体的 n 步二维路径。

示例：

```python
import numpy as np
import matplotlib.pyplot as plt
import random
```

def randomwalk2D(n):
    # [0, 0, 0, ... ,0]
    x = np.zeros(n)
    y = np.zeros(n)
    directions = ["UP", "DOWN", "LEFT", "RIGHT"]
    for i in range(1, n):
        # 随机选择一个方向
        step = random.choice(directions)
        # 根据方向移动物体
        if step == "RIGHT":
            x[i] = x[i - 1] + 1
            y[i] = y[i - 1]
        elif step == "LEFT":
            x[i] = x[i - 1] - 1
            y[i] = y[i - 1]
        elif step == "UP":
            x[i] = x[i - 1]
            y[i] = y[i - 1] + 1
        elif step == "DOWN":
            x[i] = x[i - 1]
            y[i] = y[i - 1] - 1
    # 返回物体的所有x和y位置
    return x, y

x_data, y_data = randomwalk2D(1000)
plt.title("Python中的二维随机游走")
plt.plot(x_data, y_data)
plt.show()

## 11.4 三维随机游走

既然一维和二维随机游走已经可以运行，我们现在来实现三维随机游走。

概念与二维相同，但现在你还可以额外向上、下、左、右移动，以及向内和向外移动。

示例：

```python
import matplotlib.pyplot as plt
import numpy as np
import random

def randomwalk3D(n):
    x, y, z = np.zeros(n), np.zeros(n), np.zeros(n)
    directions = ["UP", "DOWN", "LEFT", "RIGHT", "IN", "OUT"]
    for i in range(1, n):
        step = random.choice(directions)
        if step == "RIGHT":
            x[i] = x[i - 1] + 1
            y[i] = y[i - 1]
            z[i] = z[i - 1]
        elif step == "LEFT":
            x[i] = x[i - 1] - 1
            y[i] = y[i - 1]
            z[i] = z[i - 1]
        elif step == "UP":
            x[i] = x[i - 1]
            y[i] = y[i - 1] + 1
            z[i] = z[i - 1]
        elif step == "DOWN":
            x[i] = x[i - 1]
            y[i] = y[i - 1] - 1
            z[i] = z[i - 1]
        elif step == "IN":
            x[i] = x[i - 1]
            y[i] = y[i - 1]
            z[i] = z[i - 1] - 1

    return x, y, z

x_data, y_data, z_data = randomwalk3D(1000)

ax = plt.subplot(1, 1, 1, projection='3d')
ax.plot(x_data, y_data, z_data, alpha=0.9)
ax.scatter(x_data[-1], y_data[-1], z_data[-1])
plt.show()
```

```python
highest_result=die.sides+die_2.sides
for freq in range(2, highest_result+1):
```

### 13.4 Flask语法

```python
>>>from flask import Flask
```

-   Flask类的`route()`函数是一个装饰器，它告诉应用程序使用哪个URL来调用相关的函数。

-   `rule`参数是函数对URL绑定的表示。
-   `options`包含要传递给底层Rule对象的参数列表。
-   在上面的例子中，`'/'` URL被分配给了`hello world()`函数。
-   因此，当在浏览器中查看Web服务器的主页时，将渲染此函数的输出。

### 13.6 传递变量

通过在规则参数中包含可移动组件，可以动态创建 URL。该组件的变量名属性被标记。它作为关键字参数传递给与该规则关联的函数。

在下面的示例中，`route()` 装饰器的 rule 参数有一个 `<name>` 变量部分绑定到 URL "/hello"。因此，如果在浏览器中输入 URL http://localhost:5000/kiran，参数 "Kiran" 将被传递给 `hello_name()` 函数。

示例：

```python
from flask import Flask
app = Flask(__name__)
@app.route('/')
def hello_world():
    return 'Hello World'
@app.route('/python')
def hello_python():
    return 'Hello Python'
@app.route('/<name>')
def hello_name(name):
    return 'Hello %s!' % name
if __name__ == '__main__':
    app.run(debug=True)
```

## 13.7 重定向 URL

`url_for()` 函数对于为特定函数动态创建 URL 非常有用。该函数的第一个参数是函数名，它还接受一个或多个关键字参数，每个参数对应 URL 中的一个变量。

下面的脚本演示了 `url_for()` 函数的使用。

```python
from flask import Flask, redirect, url_for
app = Flask(__name__)
@app.route('/admin')
def hello_admin():
    return 'Hello Admin'
@app.route('/guest/<guest>')
def hello_guest(guest):
    return 'Hello %s as Guest' % guest
@app.route('/user/<name>')
def hello_user(name):
    if name =='admin':
        return redirect(url_for('hello_admin'))
    else:
        return redirect(url_for('hello_guest', guest = name))

if __name__ == '__main__':
    app.run(debug = True)
```

## 13.9 Flask HTTP 方法

万维网的数据通信建立在 HTTP 协议之上。该协议定义了从给定 URL 检索数据的各种技术。下表列出了各种 HTTP 方法：

### 方法与描述

| 方法 | 描述 |
| :--- | :--- |
| **GET** | 以未加密形式向服务器发送数据。最常用的方法。 |
| **HEAD** | 与 GET 相同，但没有响应体。 |
| **POST** | 用于将 HTML 表单数据发送到服务器。POST 方法接收的数据不会被服务器缓存。 |
| **PUT** | 用上传的内容替换目标资源的所有当前表示。 |
| **DELETE** | 删除由 URL 给出的目标资源的所有当前表示。 |

Flask 路由默认响应 GET 请求。但是，可以通过为 `route()` 装饰器提供 methods 参数来更改此选择。
