## 下载

### Docker

```
docker pull apachecn0/overiq-zh
docker run -tid -p <port>:80 apachecn0/overiq-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g overiq-zh
overiq-zh <port>
# 访问 http://localhost:{port} 查看文档
```
