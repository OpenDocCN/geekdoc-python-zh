## 第二章：古代文明 – Python 算术运算

欢迎来到古代文明的时代，那是数学基础奠定的时代。正如古人发明了数学一样，我们将学习 Python 如何执行数学运算。在本章中，我们将介绍算术运算、运算顺序和比较运算符。

### 算术运算

Python 支持多种算术运算。这些是你可能想执行的任何数学计算的基本构建块。让我们逐一了解它们：

#### 加法 (+)

`+` 运算符将两个数字相加。这里有一个例子：

```
##### 加法
result = 7 + 3
print(result)  # 输出：10
```

在这个例子中，Python 将 7 和 3 相加得到 10。

#### 减法 (-)

`-` 运算符从一个数字中减去另一个数字。这里有一个例子：

```
##### 减法
result = 7 - 3
print(result)  # 输出：4
```

在这个例子中，Python 从 7 中减去 3 得到 4。

#### 乘法 (*)

`*` 运算符将两个数字相乘。这里有一个例子：

```
##### 乘法
result = 7 * 3
print(result)  # 输出：21
```

在这个例子中，Python 将 7 和 3 相乘得到 21。

#### 除法 (/)

`/` 运算符将一个数字除以另一个数字。这里有一个例子：

```
##### 除法
result = 7 / 3
print(result)  # 输出：2.3333333333333335
```

在这个例子中，Python 将 7 除以 3 得到 2.3333333333333335。

#### 取模 (%)

`%` 运算符返回除法运算的余数。这里有一个例子：

```
##### 取模
result = 7 % 3
print(result)  # 输出：1
```

在这个例子中，Python 将 7 除以 3 并返回余数 1。

#### 幂运算 (**)

`**` 运算符将一个数字提升到另一个数字的幂。这里有一个例子：

```
##### 幂运算
result = 2 ** 3
print(result)  # 输出：8
```

在这个例子中，Python 将 2 的 3 次幂计算为 8。

#### 整除 (//)

`//` 运算符将一个数字除以另一个数字，并向下舍入到最接近的整数。这里有一个例子：

```
##### 整除
result = 7 // 3
print(result)  # 输出：2
```

在这个例子中，Python 将 7 除以 3 并向下舍入到 2。

### 运算顺序

就像在常规数学中一样，Python 遵循运算顺序。这个顺序通常用首字母缩写词 PEMDAS 来记忆：

-   括号
-   指数
-   乘法和除法
-   加法和减法

这里有一个例子：

```
### 运算顺序
result = (1 + 2) * 3 ** 2 / 9 - 1
print(result) # 输出：2.0
```

在这个例子中，Python 首先执行括号内的运算 (1 + 2)，然后执行指数运算 3 ** 2，接着是除法 9 / 9，最后是减法 - 1。

## 比较运算符

除了算术运算，Python 还支持比较运算符。这些运算符允许你比较两个值并返回一个布尔值（True 或 False）。以下是 Python 比较运算符的列表：

-   ==：等于
-   !=：不等于
-   <：小于
-   >：大于
-   <=：小于或等于
-   >=：大于或等于

以下是每种运算符的一个示例：

```
### 等于
print(7 == 3)  # 输出：False

### 不等于
print(7 != 3)  # 输出：True

### 小于
print(7 < 3)  # 输出：False

### 大于
print(7 > 3)  # 输出：True

### 小于或等于
print(7 <= 3)  # 输出：False

### 大于或等于
print(7 >= 3)  # 输出：True
```

## 项目：金字塔建造者

现在我们已经介绍了算术和比较运算，让我们来测试一下你的新技能。我们将创建一个程序，计算建造你自己的金字塔需要多少块积木。

假设我们想建造一个底座为 10 块乘以 10 块的金字塔。我们可以通过将每一层的积木数量相加来计算所需的总积木数。底层有 10 * 10 = 100 块积木，下一层有 9 * 9 = 81 块积木，依此类推，直到顶层，有 1 * 1 = 1 块积木。

以下是我们如何在 Python 中计算这个：

```
### 底座大小
base = 10

### 总积木数
total_blocks = 0

### 计算每一层的积木数
for i in range(base, 0, -1):
    layer_blocks = i * i
    total_blocks += layer_blocks

### 打印总积木数
print("You would need", total_blocks, "blocks to build your pyramid.")
```

当你运行这段代码时，你应该会看到消息 “You would need 385 blocks to build your pyramid.” 被打印到屏幕上。

## 字符串

Python中的字符串是字符的序列。Python中的字符串是不可变的，这意味着它们在创建后无法更改。字符串可以通过将字符括在单引号或双引号内来创建。

#### 创建字符串

以下是在Python中创建字符串的方法：

```python
#### 创建字符串
my_string = "Hello, World!"
print(my_string)  # 输出：Hello, World!
```

在这个例子中，我们创建了一个包含文本"Hello, World!"的字符串`my_string`。

### 字符串连接

你可以使用+运算符将两个或多个字符串组合在一起。这被称为字符串连接。

```python
### 字符串连接
greeting = "Hello"
name = "Alice"
message = greeting + ", " + name + "!"
print(message)  # 输出：Hello, Alice!
```

在这个例子中，我们将字符串greeting、一个逗号和一个空格、name以及一个感叹号组合起来，创建了消息"Hello, Alice!"。

### 字符串长度

你可以使用`len()`函数来查找字符串的长度。

```python
### 字符串长度
my_string = "Hello, World!"
print(len(my_string))  # 输出：13
```

在这个例子中，`len()`函数返回`my_string`的长度，即13。

### 字符串索引

你可以使用索引来访问字符串中的单个字符。Python中的索引从0开始。

```python
### 字符串索引
my_string = "Hello, World!"
print(my_string[0])  # 输出：H
print(my_string[7])  # 输出：W
```

在这个例子中，`my_string[0]`返回`my_string`的第一个字符，即H，而`my_string[7]`返回第八个字符，即W。

### 字符串切片

你可以通过指定索引范围来提取字符串的一部分（一个“切片”）。

```python
### 字符串切片
my_string = "Hello, World!"
print(my_string[0:5])  # 输出：Hello
print(my_string[7:12])  # 输出：World
```

在这个例子中，`my_string[0:5]`返回`my_string`的前五个字符，即Hello，而`my_string[7:12]`返回从第八个字符开始的五个字符，即World。

## 列表

Python中的列表是项目的有序集合。列表是可变的，这意味着它们在创建后可以更改。列表可以包含不同类型的项目。

#### 创建列表

以下是在Python中创建列表的方法：

```python
#### 创建列表
my_list = [1, 2, 3, 4, 5]
print(my_list)  # 输出：[1, 2, 3, 4, 5]
```

在这个例子中，我们创建了一个包含数字1到5的列表`my_list`。

#### 列表长度

你可以使用`len()`函数来查找列表的长度。

```python
#### 列表长度
my_list = [1, 2, 3, 4, 5]
print(len(my_list))  # 输出：5
```

在这个例子中，`len()`函数返回`my_list`的长度，即5。

#### 列表索引

你可以使用索引来访问列表中的单个项目。

```python
#### 列表索引
my_list = [1, 2, 3, 4, 5]
print(my_list[0])  # 输出：1
print(my_list[2])  # 输出：3
```

def caesar_cipher(message, shift):
    alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    encrypted_message = ""
    for char in message:
        if char in alphabet:
            index = (alphabet.index(char) + shift) % 26
            encrypted_message += alphabet[index]
        else:
            encrypted_message += char
    return encrypted_message

### 循环

在 Python 中，循环用于重复执行一段代码块，直到满足某个条件。Python 中有两种类型的循环：`for` 和 `while`。

#### For 循环

`for` 循环用于遍历一个序列（可以是列表、元组、字典、集合或字符串）或其他可迭代对象。遍历序列称为遍历。

以下是一个示例：

```python
#### For 循环
for i in range(5):
    print(i)
```

在此示例中，`for` 循环打印从 0 到 4 的数字。`range(5)` 函数生成一个从 0 到 4 的数字序列，for 循环遍历此序列，依次打印每个数字。

#### While 循环

Python 中的 while 循环用于只要给定条件为真，就重复执行一段语句块。条件在执行循环体之前进行检查。

以下是一个示例：

```python
#### While 循环
i = 0
while i < 5:
    print(i)
    i += 1
```

在此示例中，while 循环打印从 0 到 4 的数字。只要条件 i < 5 为真，循环就会继续执行。在循环内部，我们在每次迭代后将 i 的值增加 1。

### 循环控制语句

循环控制语句改变循环的正常执行顺序。Python 支持以下控制语句：break 和 continue。

#### Break 语句

Python 中的 break 语句终止当前循环，并在下一条语句处恢复执行，就像 C 语言中的传统 break 一样。

以下是一个示例：

```python
#### Break 语句
for i in range(5):
    if i == 3:
        break
    print(i)
```

在此示例中，for 循环开始打印从 0 到 4 的数字。但是，当 i 等于 3 时，执行 break 语句，循环终止。因此，打印了数字 0、1 和 2，但没有打印 3 和 4。

#### Continue 语句

Python 中的 continue 语句将控制权返回到循环的开头。continue 语句拒绝当前迭代循环中的所有剩余语句，并将控制权移回循环的顶部。

以下是一个示例：

```python
#### Continue 语句
for i in range(5):
    if i == 3:
        continue
    print(i)
```

在此示例中，for 循环开始打印从 0 到 4 的数字。但是，当 i 等于 3 时，执行 continue 语句，跳过循环的当前迭代。因此，打印了数字 0、1、2 和 4，但没有打印 3。

file = open('example.txt', 'r')

以下是如何关闭文件的方法：

file.close()
```

def translate_to_morse(message):
    morse_code = {
        'A': '.-', 'B': '-...', 'C': '-.-.', 'D': '-..',
        'E': '.', 'F': '..-.',
        'G': '--.', 'H': '....', 'I': '..', 'J': '.---',

        'M': '--', 'N': '-.', 'O': '---', 'P': '.--.', 'Q': '--.-', 'R': '.-.',
        'S': '...', 'T': '-', 'U': '..-', 'V': '...-', 'W': '.--', 'X': '-..-',
        'Y': '-.--', 'Z': '--..'
    }
    morse_message = ''
    for char in message:
        if char.upper() in morse_code:
            morse_message += morse_code[char.upper()] + '\n'
        else:
            morse_message += ' '
    return morse_message

def send_morse_message(input_file, output_file):
    file = open(input_file, 'r')
    message = file.read()
    file.close()

    morse_message = translate_to_morse(message)

    file = open(output_file, 'w')
    file.write(morse_message)
    file.close()
```

## 错误处理

在 Python 中，有两种类型的错误：语法错误和异常。

### 语法错误

语法错误，也称为解析错误，发生在 Python 解析器无法理解一行代码时。这些错误可能是最容易发现和修复的。它们通常发生在你打错字、忘记冒号或在代码中犯了其他语法错误时。

以下是一个语法错误的示例：

```
print("Hello, world!)
```

在这个例子中，我们忘记了关闭字符串周围的引号。Python 将引发一个 `SyntaxError` 并指出代码中它感到困惑的位置。

## 异常

即使你的代码在语法上是正确的，在执行时仍然可能导致错误。这些错误被称为异常。它们发生在 Python 解释器遇到它不知道如何处理的情况时。

以下是一个异常的示例：

```
print(10 / 0)
```

在这个例子中，我们试图除以零，这在数学上是未定义的。当 Python 尝试执行这一行时，它将引发一个 `ZeroDivisionError`。

## Try, Except 块

在 Python 中，我们可以使用 `try` 和 `except` 块来捕获和处理异常。我们将可能引发异常的代码放在 `try` 块中，将处理异常的代码放在 `except` 块中。

以下是一个示例：

```
try:
    print(10 / 0)
except ZeroDivisionError:
    print("You can't divide by zero!")
```

在这个例子中，当 Python 遇到 `ZeroDivisionError` 时，它会执行 `except` 块并打印 "You can't divide by zero!"，而不是使程序崩溃。

你可以使用多个 `except` 块来捕获多个异常：

```
try:
    print(10 / 'five')
except ZeroDivisionError:
    print("You can't divide by zero!")
except TypeError:
    print("You can't divide a number by a string!")
```

在这个例子中，我们试图将一个数字除以一个字符串，这会引发一个 TypeError。Python 捕获这个异常并打印 "You can't divide a number by a string!"。

## 引发异常

有时，你可能希望在满足特定条件时引发异常。你可以使用 raise 语句来做到这一点。

以下是一个示例：

```
def divide(x, y):
    if y == 0:
        raise ZeroDivisionError("You can't divide by zero!")
    return x / y
```

在这个例子中，我们定义了一个函数 divide，它接受两个参数 x 和 y，并返回 x 除以 y 的结果。如果 y 为零，我们引发一个带有自定义错误消息的 ZeroDivisionError。
