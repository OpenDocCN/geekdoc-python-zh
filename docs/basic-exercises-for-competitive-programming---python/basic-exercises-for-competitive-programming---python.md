## 练习 1

编写一个程序，读取一个字符串 S，并在标准输出中以相反的顺序（即从右到左翻转）打印该行。

#### 输入

唯一的输入是一个字符串 S。

#### 输出

以相反的顺序打印字符串。

### 示例

#### 输入

```
1 2 3 hello
```

#### 输出

```
olleh 3 2 1
```

## 解决方案

解决此任务的一种简单方法是使用循环从最后一个位置到第一个位置打印字符。这避免了保存反转列表的需要，只需打印每个字符即可。

用Python编写的解决方案代码如下所示。

```python
l = input()
i = len(l)
while i > 0:
    i = i - 1
    print(l[i], end='')
print()
```

## 练习 2

给定一个正整数 n 作为输入。如果 n 是偶数，算法将其除以二；如果 n 是奇数，算法将其乘以三并加一。算法重复此过程，直到 n 等于一。例如，当 n=5 时，序列如下：
5→16→8→4→2→1
你的任务是模拟算法对给定值 n 的执行过程。

#### 输入

唯一的输入行包含一个正整数 n。

#### 输出

打印一行，包含所有用空格分隔的 n 值。

### 示例

#### 输入

```
5
```

#### 输出

```
5 16 8 4 2 1
```

## 解决方案

为了解决此任务，使用一个循环来验证数字是否尚未为1。在循环内，检查数字是否为偶数，如果是，则除以二；否则，将数字乘以3并加1。打印数字，如此重复直到数字为1。

用Python编写的解决方案代码如下所示。

```python
n = int(input())
print(n, end=' ')
while n > 1:
    if n % 2 == 0:
        n //= 2
    else:
        n = 3 * n + 1
    print(n, end=' ')
```

## 练习 4

给定从1,2,...,n中除一个之外的所有数字。你的任务是找出缺失的数字。

#### 输入

第一行输入包含一个整数 n。
第二行包含 n−1 个数字。每个数字都是唯一的，且在1到n之间（包含1和n）。

#### 输出

打印缺失的数字。

### 示例

#### 输入

```
10
2 8 10 6 5 1 3 7 4
```

#### 输出

```
9
```

## 解决方案

为了解决此任务，首先保存整数 n。然后保存整数行。初始化一个和，将每个整数相加。最后，使用高斯求和公式：n * (n + 1) / 2。减去之前计算的和，即可得到缺失的数字。这是可行的，因为高斯求和根据数字总数得到真实的总和，如果减去所提供数字的和，结果将显示缺失的值，从而得到真实的总和。

用Python编写的解决方案代码如下所示。

```python
n = int(input())
arr = map(int, input().split())
s = 0

for a in arr:
    s += a

print(n * (n + 1) // 2 - s)
```

## 练习 5

给定一个整数序列 A，一个最大相同序列是具有最大长度 k 的相邻相同值的序列。编写一个程序，读取一个输入序列 A（每行一个或多个数字），并输出 A 中第一个（最左边的）最大相同序列的值。

#### 输入

第一行输入包含一个整数 n。
第二行包含 n 个整数。

#### 输出

打印一个整数：最长重复的长度。

### 示例

#### 输入

```
2 4 6 2 2
```

#### 输出

```
2
```

## 解决方案

为了解决此任务，了解哪个数字重复次数最多的一种有效方法是使用变量来帮助我们完成过程。在这种情况下，使用 'k' 来确定一个数字重复了多少次，'i' 保存前一个索引，'j' 保存当前索引，'v' 存储顺序重复最多的数字。

在循环开始时，比较前一个值和当前值，如果不同，则交换索引。索引 'j' 增加，并比较重复的数字是否大于 'k'，如果是，则 'v' 取该数字的值，'k' 取该数字重复的次数。

用Python编写的解决方案代码如下所示。

```python
A = list(map(int, input().split()))

v = A[0]
k = 1
i = 0
j = 1
while j < len(A):
    if A[j] != A[i]:
        i = j
    j += 1
    if j - i > k:
        v = A[i]
        k = j - i

print(v)
```

## 练习 6

给定一个DNA序列：一个由字符 A、C、G 和 T 组成的字符串。你的任务是找出序列中最长的重复。这是一个只包含一种字符类型的最长子串。

#### 输入

唯一的输入行包含一个由 n 个字符组成的字符串。

#### 输出

打印一个整数：最长重复的长度。

### 示例

#### 输入

```
ATTGCCCA
```

#### 输出

```
3
```

## 解决方案

要解决此任务，需要获取输入字符串，使用变量 `ans` 存储响应，`count` 计数重复次数，`l` 包含要比较的字符。使用循环遍历字符串中的每个字符，如果字符重复，则增加计数器，并将 `ans` 替换为 `count` 和 `ans` 之间的最大值，否则 `l` 变为新字符，计数器重置。

以下是用 Python 编写的解决方案代码。

```
s = input()
ans = 1
count = 0
l = 'A'
for cha in s:
    if cha == l:
        count += 1
        if count > ans:
            ans = count
    else:
        l = cha
        count = 1
print(ans, end=' ')
```

## 练习 8

给定一个包含 n 个整数的数组。你希望修改数组使其递增。每个元素至少与前一个元素一样大。

每次移动，你可以将任何元素的值增加一。你的任务是找到所需的最小移动次数。

#### 输入

第一行输入包含一个整数 n，即数组的大小。
然后，第二行包含 n 个整数，即数组的内容。

#### 输出

打印最小移动次数。

### 示例

#### 输入

```
5

2 1 4 1 5
```

#### 输出

```
4
```

## 解决方案

要解决此任务，我们必须找到增加数组值以使列表递增所需的最小移动次数。读取输入数据后，初始化变量 `mx`（最大值）和 `ans`（存储为零），遍历数组中的每个数据，查找最大值并减去数字的值，将结果保存在响应中。这之所以有效，是因为它检查当前数字之前的数字是否更大，获取当前与之前的差值，换句话说，它获取使两个值相等所需的增加量。

以下是用 Python 编写的解决方案代码。

```
n = int(input())
arr = map(int, input().split())
mx = 0
ans = 0

for x in arr:
    mx = max(x, mx)
    ans += mx - x

print(ans)
```

#### 输出

打印整数 1,2,...,n 的一个漂亮排列。如果有多个解决方案，你可以打印其中任何一个。如果没有解决方案，打印“NO SOLUTION”。

## 解决方案

要解决此任务，要求元素之间的差不等于 1。当 n 等于 1 时，打印 1；如果小于 4，则没有解决方案；在所有其他情况下，只需从 2 开始以 2 为步长打印到 n+1 的值，在另一个循环中使用相同的策略但从 1 开始。

```
n = int(input())
if n == 1:
    print(1, end=' ')
elif n < 4:
    print('NO SOLUTION')

else:
    for i in range(2, n+1, 2):
        print(i, end=' ')

    for i in range(1, n+1, 2):
        print(i, end=' ')
```

## 练习 10

数字螺旋是一个无限网格，其左上角方格的数字为 1。以下是螺旋的前五层：

| 1 | 2 | 9 | 10 | 25 |
|---|---|---|---|---|
| 4 | 3 | 8 | 11 | 24 |
| 5 | 6 | 7 | 12 | 23 |
| 16 | 15 | 14 | 13 | 22 |
| 17 | 18 | 19 | 20 | 21 |

你的任务是找出第 x 行和第 y 列的数字。

#### 输入

第一行输入包含一个整数 t，即测试用例的数量。
之后，有 t 行，每行包含整数 x 和 y。

#### 输出

对于每个测试用例，打印第 x 行和第 y 列的数字。

### 示例

#### 输入

2

3 4

4 1

#### 输出

12

16

## 解决方案

要解决此任务，不需要创建包含无限网格的矩阵。首先存储输入值，然后对于每个 'x'、'y'，获取最大值并将其保存在 'z' 中，在 'z2' 中保存较大数字的前一个数字与自身的乘积。如果 `z` 是奇数，且 `x` 是最大值，则只需将 `y` 加到 `z2` 上；否则将 `2*z - x` 加到 `z2` 上。如果 `z` 是偶数，且 `y` 是最大值，则只需将 `x` 加到 `z2` 上；否则将 `2*z - y` 加到 `z2` 上。

以下是用 Python 编写的解决方案代码。

```
t = int(input())
for i in range(t):
    x, y = map(int, input().split())
    z = max(x, y)
    z2 = (z - 1) * (z - 1)
    if z % 2 != 0:
        if x == z:
            ans = z2 + y
        else:
            ans = z2 + 2 * z - x
    else:
        if y == z:
            ans = z2 + x
        else:
            ans = z2 + 2 * z - y
    print(ans)
```

|---|---|---|---|---|---|
| 2 | 3 | 4 | 4 | 3*3 = 9 | 9 + 3 = 12 |
| | 4 | 1 | 4 | 3*3 = 9 | 9 + 2 * 4 - 1 = 16 |

```python
    return True
```

## 练习 12

你的任务是计算对于 k=1,2,...,n，在 k×k 的棋盘上放置两个骑士，使它们不互相攻击的方式数量。

为了解决这个任务，首先需要获取它们被攻击时的所有可能位置。存储整数 'n'。对于每个 'k'（矩阵的边长），从1到 'n + 1'，'a1' 存储棋盘上的总格子数，'a2' 存储它们可以在棋盘上移动的总方式数。当棋盘边长为3或更大时，如果两个骑士互相攻击，它们将位于一个 2 * 3 或 3 * 2 的矩形内。因此，放置它们的方式数为 (n-1) * (n-2) + (n-2) * (n-1)。骑士互相攻击的总位置数为 2 * 2 * (n-1) * (n-2) = 4 * (n-1) * (n-2)。必须从两个骑士的总移动方式中减去这个值，才能得到它们不互相攻击的移动方式数。

以下是用Python编写的解决方案代码。

```python
n = int(input())
for k in range(1, n + 1):
    a1 = k * k

    if k > 2:
```

## 解决方案

要解决此任务，请使用 `itertools` 库中的 `combinations` 方法。第一个 for 循环控制每个集合的长度，而第二个循环遍历每种可能的组合，寻找两个集合之间的最小差值。首先 `w1` 计算集合的和，`w2` 减去重量的和，如果两者的差值小于 `ans`，则替换 `ans`，继续直到找到最小的可能差值。

以下是用 Python 编写的解决方案代码。

```python
from itertools import combinations
n = int(input())
weights = [int(i) for i in input().split()]
ans = sum(weights)
for i in range(1,n):
    for comb in combinations(weights,i):
        w1 = sum(comb)
        w2 = sum(weights)-sum(comb)
        if abs(w2-w1)<ans:
           ans = abs(w2-w1)
print(ans)
```

要解决此任务，首先将矩阵中的 '.' 替换为 0，将 '*' 替换为 1。接下来，初始化长度为 8 的变量列 'col'，长度为 15 的对角线 'diag1' 和 `diag2`，以及初始值为 0 的 'count'。使用递归方法遍历棋盘，当 'n' 达到 8 时，意味着可以进行一次移动，因此 'count' 增加。循环用于遍历每个'保留'位置，只要它不是保留的，并且列和对角线都保持为 True，就更改值以确保两个皇后不会接触。

以下是用 Python 编写的解决方案代码。

```python
reserved = [[0]*8 for _ in range(8)]
for i in range(8):
    row = list(str(input()))
    for j in range(8):
        if row[j] == '*':
            reserved[i][j] = 1

col = [True] * 8
diag1 = [True] * 15
diag2 = [True] * 15
count = 0
def chess(n):
    global count
    if n == 8:
        count += 1
        return
    for m in range(8):
        if reserved[n][m] == 0 and col[m] and diag1[m+n] and diag2[7-n+m]:
            col[m] = diag1[m+n] = diag2[7-n+m] = False
            chess(n+1)
            col[m] = diag1[m+n] = diag2[7-n+m] = True

chess(0)
print(count)
```

## 练习 23

编写一个程序，从标准输入读取一个单词列表（每行一个单词），并打印出出现次数最少的单词中最短的任意一个。

第一行包含一个整数 n，表示单词的数量。
每行包含一个单词。

#### 输出

打印出出现次数最少的单词中最短的任意一个。

### 示例

#### 输入

```
4
water
sand
water
beach
```

#### 输出

```
sand
```

## 解决方案

要解决此任务，读取每一行并将其存储在字典中以计算频率。变量 'min_v' 用于保存最低频率，'min_len_word' 用于保存最短长度的单词。由于初始值为 'None'，它们从字典的第一个值开始，否则如果值小于 'min_v'，则替换单词和值，最后如果频率相同但单词长度小于 'min_len_word'，则替换它。

以下是用 Python 编写的解决方案代码。

```python
D = {}
n = int(input())
for w in range(n):
    w = input().strip()
    if w in D:
        D[w] += 1
    else:
        D[w] = 1

min_v = None
min_len_word = None
for k,v in D.items():
    if min_v == None:
        min_v = v
        min_len_word = k
    elif v < min_v:
        min_v = v
        min_len_word = k
    elif v == min_v and len(k) < len(min_len_word):
        min_len_word = k

print(min_len_word)
```

## 练习 24

编写一个程序，读取一个数字字符串 S 作为命令行参数，并在数字序列中插入加号（+）和减号（-），以创建一个计算结果为零的表达式。程序应输出结果表达式（如果存在这样的表达式），或者输出 "None"（如果不存在这样的表达式）。

#### 输入

唯一的输入行是一个数字字符串 S。

#### 输出

打印结果表达式（如果存在这样的表达式），或者输出 "None"（如果不存在这样的表达式）。

### 示例

#### 输入

132

#### 输出

+1-3+2

## 解决方案

为了解决这个任务，我们使用递归方法遍历字符串以找到合适的符号。该方法设有条件，以便在递归发生时能够退出。我们启动一个循环，从中提取“前部”、“后部”和“目标值”，通过这种方式递归调用该方法，返回带有相应符号的字符串部分。

以下是用Python编写的解决方案代码。

```python
def solve(s, n):
    v = int(s)
    if v == n:
        return '+' + s

    if v == -n:
        return '-' + s

    if v < abs(n):
        return None

    for i in range(1, len(s)):
        front = s[0:i]
        back = s[i:]
        target = int(front)
        back_solution = solve(back, n-target)
        if back_solution != None:
            return '+' + front + back_solution
        back_solution = solve(back, n+target)
        if back_solution != None:
            return '-' + front + back_solution
    return None

print(solve(input(), 0))
```
