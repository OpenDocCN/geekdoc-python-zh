# 第三章

## Python 入门

既然我们已经对 Python 及其能做什么有了一些了解，让我们来看看安装 Python 可以采取的一些步骤。Python 兼容三大操作系统：Linux、Mac OS X 和 Windows。无论你在过程中偏好哪种类型的系统，这都使其更易于使用和操作。以下是在系统上安装 Python 可以采取的一些步骤：

## Mac OS X 入门

如果你使用 Mac 电脑，你应该能找到已安装的 Python 2。你使用的 Python 2 版本会因电脑的年龄和所使用的操作系统而异。但是，Python 应该以某种形式存在。要查明你的电脑上安装了哪个 Python 2 版本，请打开终端应用程序并输入以下命令：

```
python –V
```

这将通过显示一个数字来显示你将获得的版本。如果你愿意，你可以在此系统上安装 Python 3 而不是 2.X 版本，并且你不必卸载 2.X 版本。要检查 3.X 安装，只需启动终端应用程序并输入以下命令：

```
Python3 –V
```

对于这个命令，请记住，此类操作系统的默认设置是 Python 3 根本不会被安装。这意味着如果你想使用更新版本的 Python，例如 Python 3，你必须访问 www.python.org 并下载你想要的 Python 版本。还有其他一些地方可以获取 Python，但上面列出的网站可以为你完成所有操作，并确保你免费获得所需的 IDLE、解释器和其他工具。

一旦你开始用这种语言编写代码，你就会发现安装 Python Shell 和 IDLE 对于编写代码至关重要。你应该仔细检查这两者是否都已安装在你的系统上，最好的方法是使用以下说明：

- 你可以在 Python 2 中输入 "Idle"。
- 你可以在 Python 3.X 中输入 "idle3."。

当你安装 Python 3 时，IDLE 将作为标准应用程序的一部分包含在内。正如我们之前提到的，在安装 Python 3 时，你需要将 IDLE 作为常用应用程序的一部分进行安装。然后，这部分代码将位于应用程序文件夹中。要启动此程序并在桌面上工作，只需打开文件夹，双击 IDLE 应用程序，然后开始工作。

## Windows 入门

在处理一些 Python 需求时，除了使用 Mac 操作系统外，你还可以在 Windows 电脑上下载和安装这种语言。而且，由于许多人喜欢使用这种编程语言，因为它易于使用且功能广泛，知道 Python 也能在 Windows 上流畅运行是令人欣慰的。

在使用 Windows 操作系统时，需要注意的一点是 Python 不会预装。这是因为 Windows 已经有自己的编码系统，不会添加另一个。好消息是 Python 将继续正常运行。我们只是认为没有必要在系统上预装并准备好使用它。

在 Windows 电脑上安装 Python 不一定很复杂。你可以采取以下步骤确保其安装：

- 1. 首先，你需要访问官方 Python 下载页面并下载 Windows 安装程序。你可以下载最新版本的 Python 3 或选择其他选项。安装程序默认将安装 32 位版本的 Python，但如果你愿意，可以升级到 64 位版本。通常建议使用 32 位版本以确保与旧软件包没有兼容性问题，但你可以根据需要进行尝试。
- 2. 右键单击安装程序，从上下文菜单中选择“以管理员身份运行”。将有两个选项供你选择。你应该选择“自定义安装”。
- 3. 在下一个屏幕上，确保“可选功能”下的所有复选框都已选中，然后单击继续。
- 4. 你应该在“高级选项”下选择你希望安装 Python 的位置。选择所需的安装位置。留出一些时间让其完成，然后关闭安装程序。
- 5. 接下来，修改系统的 PATH 变量以包含包含你稍后需要的软件包和其他组件的目录。按照以下步骤操作：
    - 要访问控制面板，请转到“开始”菜单并选择“控制面板”。为此，右键单击任务栏并输入“控制面板”。选择该图标。
    - 在控制面板中查找“环境”。然后从下拉菜单中选择“编辑系统环境变量”。然后你可以从下拉菜单中选择“环境变量”。
    - 转到“用户变量”部分。你可以编辑现有的 PATH 变量或创建一个新的。
    - 如果系统没有 PATH 变量，请单击“新建”创建一个。为 PATH 变量命名，并将其包含在你希望的文件夹中。关闭所有控制面板对话框并继续。
- 6. 现在你可以使用命令提示符了。为此，请转到“开始”菜单，然后转到“Windows 系统”，最后转到“命令提示符”。在搜索框中输入“python”。这将为你启动 Python 解释器。

从这里开始，该程序将准备好在我们的 Windows 系统上使用。你可以选择通过打开所有不同的部分来查看它们。在开始你的编码之旅之前，请先研究可用的功能！

## Linux 入门

在 Linux 操作系统上使用 Python 编程语言同样具有优势。虽然 Linux 操作系统可能不如 Windows 和 Mac 使用广泛，但在这个系统上使用 Python 仍有许多优点。Linux 是编程和其他任务的绝佳选择。

如果你想使用 Linux 操作系统，你应该先检查你的系统是否已经安装了 Python 版本。这很简单，你只需要使用以下代码即可开始：

```
$ python3 --version
```

在 Ubuntu 16.10 或更高版本上安装 Python 3.6 是一个简单的过程。你只需输入以下命令：

```
$ sudo apt-get update
$ sudo apt-get install python3.6
```

如果你使用的是旧版本的 Ubuntu 或其他操作系统，你需要使用 dead snakes PPA 或其他工具来获取 Python 3.6 版本。以下是执行此操作所需的代码：

```
$ sudo apt-get install software-properties-common
$ sudo add-apt-repository ppa:deadsnakes/ppa
$ sudo apt-get update
$ sudo apt-get install python3.6
```

好消息是，如果你使用的是其他 Linux 发行版，Python 3 几乎肯定已经安装。如果没有，你可以使用发行版附带的包管理器。如果 Python 3 包不是最新的或不是你需要的正确版本，你可以按照这些相同的步骤来帮助在你的计算机上安装更新版本的 Python。

# 第 4 章

## Python 基础

熟悉 Python 编程语言最有效的方法之一就是亲自尝试。你练习用这种语言编写一些代码的次数越多，你就会对它越熟悉，从而更容易处理一些更复杂的内容。这就是为什么我们将在本章花一些时间来讲解 Python 编程的基础知识，以及如何使用你的 IDE 和编译器来完成工作。

在本章中，我们将专注于编写我们的前几行代码。对于这一章，我们将保持简单，学习最基本的代码之一：Hello, World！

好消息是，我们可以将许多不同的部分与这段代码结合起来，以达到最佳效果。随着我们逐步深入这本指南，我们只需要在进入稍微复杂的内容之前学习基础知识。

你可以使用你的命令行文本编辑器来处理 Hello World! 程序。这应该随你下载的 Python 版本一起提供，所以打开它并开始处理一个新文件。在信封内写下以下行：

```
$ nano hello.py
```

在你的终端窗口中打开此文本文件后，输入以下行来运行程序：

```
print("Hello, World!")
```

现在让我们来剖析这段代码的各个组成部分。`print()` 函数用于指示计算机执行特定任务。因为它有括号，所以我们知道它是一个函数。这个函数要求编译器将我们输入在括号内的任何内容作为输出打印出来。默认情况下，这将输出到当前终端窗口。

一些函数，如 `print()`，将默认包含在 Python 语言中。你可以在任何你想要编写的程序中使用它们。你也可以使用一些其他元素来创建你自己的函数。

现在，当你处理这个函数的括号时，你会注意到我们添加了 "Hello, World!" 这个术语。为了使其工作，这将被括在引号中。我们使用的字符串将是引号中的任何字符。一旦我们为程序写下了这个，我们可以通过按 X 键退出代码。当出现提示时，按 y 返回 shell 并继续下一步。

## 运行你的程序

你现在可以在你编写它的程序中运行这个程序。我们将使用 `python3` 命令与我们给程序文件的名称一起使用。只需在命令提示符下输入以下命令即可启动程序：

```
$ python3 hello.py
```

当你在命令行中输入此命令时，终端将显示以下输出：

```
Hello, World!
```

让我们仔细看看这一过程中发生了什么。Python 使用 `print()` 函数来执行我们之前讨论的 `print("Hello, World!")` 这一行。然后，该过程被传递了字符串值。

因为该值将被传递给函数，所以代码中的 Hello, World 部分在此示例中将被称为参数。语句两侧的引号不会出现在屏幕上，因为它们的唯一目的是告知 Python 编译器该函数将包含一个字符串。

这些引号不是字符串的基本部分，它们将标记字符串的开始和结束位置。

## 需要了解的代码部分

既然我们有机会查看 Python 代码以及如何使用它，现在是时候专注于学习这种类型代码附带的一些基本组成部分了。Python 编码不应该很困难，但理解一些关键特性以及它们如何协同工作可以对你的整体编码能力产生重大影响。

处理文件将是Python编程的一个关键方面。文件是磁盘上的一个命名位置，用于存储相关数据。它将用于将你需要处理的一些数据永久保存在非易失性内存或硬盘中。因为RAM（随机存取存储器）非常易失，当机器关闭时它会丢失数据。我们希望处理文件，因为它们确保我们将来可以找到我们的信息。

## 打开文件

Python 被设计成内置了一个函数来帮助你打开想要的文件。这个函数被称为 `open()`。这个函数会返回我们所需的文件对象或句柄，因为它被用来帮助你按照期望的方式读取或修改文件。当我们打开文件时，可以指定模式。我们必须指定是使用 "r" 进行读取、"w" 进行写入，还是 "a" 进行追加。此外，我们必须指定是以二进制模式还是文本模式打开文件。

默认情况下，文件将以文本模式打开。当我们在此模式下从文件读取时，会得到一些字符串作为结果。不过，你有多种打开文件的选项，包括以二进制模式打开。当你选择二进制模式时，我们将收到字节，这在处理包含文本的文件时可能很有用。

我们需要了解一些可用的不同 Python 文件模式来帮助理解这一点。以下是一些选项，可以帮助你以正确的顺序和方式打开想要的文件：

- 1. `r`：这将帮助我们读取文件。
- 2. `w`：这将打开一个我们可以写入的文件。如果文件不存在，它会创建一个新文件；如果文件已存在，则会截断该文件。
- 3. `x`：这将帮助我们以独占创建方式打开文件。如果文件已存在，操作将失败。
- 4. `a`：这将打开文件以进行追加。这允许你在不截断文件的情况下向文件末尾添加额外数据。如果文件不存在，也可以用它来创建新文件。
- 5. `t`：这将以文本模式打开文件。
- 6. `b`：文件将以二进制模式打开。
- 7. `+`：无论你是读取还是写入文件，这都会打开它以进行更新。

## 如何写入文件

接下来我们要看的是使用 Python 创建文件。我们有几种写入文件的方式，包括写入、独占创建和追加模式。请谨慎使用写入模式，因为它会覆盖你在文件中之前完成的所有工作。此选项会清除所有先前的数据，因此请仔细确认这是你希望执行的操作。

`write()` 方法用于写出字符串，如果你处理的是二进制文件，则是字节序列。这是一种可以告诉我们文件中有多少字符的方法。你可以通过以下代码看到这一点：

## 元组

Python 元组是本指南中我们需要花时间研究的下一个主题。当我们谈论 Python 中的元组时，我们谈论的是一个列表。然而，在开始之前，有一些区别需要考虑。列表和元组之间的主要区别在于，一旦元组的元素被赋值，就不能更改。然而，在使用列表时，你可以在代码中交互时更改组件。

另一方面，列表和元组看起来几乎相同。这取决于你是否希望在元素赋值后能够更改它们。如果你想能够进行修改，就需要使用列表。如果你想在元素赋值后保持它们完全相同，就需要使用元组。

当我们希望使用元组时，我们将研究的最关键任务是如何创建自己的元组。这是一个简单的方法，你可以通过将所有元素或项目放在括号内并用逗号分隔来创建它。请记住，这些括号是可选的，但包含它们通常被认为是良好的实践，并且使代码更易于理解。

在处理元组时，你会注意到你可以包含任意多的项目。几乎总是会有两个，因为你正在处理一个想法列表，但你不必担心最大值或最小值。你还可以混合搭配不同类型的项目，因此你可以包含任意数量的字符串、列表、浮点数和整数。

既然我们已经讨论了元组以及它们与你正在编写的某些 Python 代码的关联，现在是时候看看创建元组所需的一些代码了。以下是将帮助你创建元组的 Python 代码示例。

```python
# 空元组
my_tuple = ()
print(my_tuple)  # 输出：()

# 包含整数的元组
my_tuple = (1, 2, 3)
print(my_tuple)  # 输出：(1, 2, 3)

# 包含混合数据类型的元组
my_tuple = (1, "Hello", 3.4)
print(my_tuple)  # 输出：(1, "Hello", 3.4)

# 嵌套元组
my_tuple = ("mouse", [8, 4, 6], (1, 2, 3))
print(my_tuple)  # 输出：("mouse", [8, 4, 6], (1, 2, 3))
```

考虑到这一点，你会发现使用元组而不是列表有各种优势。因为这些元组类似于列表并在类似的情况下使用，所以很难确定何时应该使用一个，以及为什么元组比使用列表有一些优势。在代码中使用元组而不是列表的一些最显著优势包括：

- 1. 元组将用于异构或各种形式的数据以及同构或相似类型的数据。
- 2. 因为这些元组被认为是不可变的，所以遍历它们通常比遍历列表更快。这使我们能够在代码中使用元组时获得轻微的性能提升。
- 3. 元组将包含一些字典的不可变元素，这些元素可以用作键。使用列表时，这是不可能的。
- 4. 如果你有不会更改的数据，将其作为元组实现将帮助你保护它。

# 章节总结

元组和列表非常相似，如果你查看代码，它们看起来会是一样的。主要区别在于你是否希望在元素赋值后允许编辑项目列表。一旦你为元组赋值，它将保持不变，你将无法更改它。但是，你可以在任务完成后使用列表进行一些必要的调整。理解这两者如何交互会影响你可以编写的代码类型。

你可能还会注意到这个猜词游戏包含一些条件语句。条件语句在这个游戏中至关重要，因为它们会告诉我们选择的字母对于该单词是否正确。而且，对于我们要构建的代码，你不需要负责找出秘密单词，因为 Python 会为我们完成。这让你实际上可以玩游戏。我们可以使用条件语句在你猜对时显示正确的字母，或者在你选择错误时通知你。

```python
#importing the time module
import time

#welcoming the user
print("Hello, Time to play hangman!")

#wait for 1 second
time.sleep(1)

print("Start guessing...")
time.sleep(0.5)

#here, we set the secret word
word = "secret"

#creates a variable with an empty value
guesses = ""

#determine the number of turns
turns = 10

# Create a while loop
#check if the turns are more than zero
while turns > 0:
    #    make a counter that starts with zero
    failed = 0

    #    for every character in secret_word
    for char in word:
        #    see if the character is in the players guess
        if char in guesses:
            #    print then out the character
            print(char, end="")
        else:
            #    if not found, print a dash
            print("_", end="")
            #    and increase the failed counter with one
            failed += 1

    #    if failed is equal to zero
    if failed == 0:
        # print You Won
        print("\nYou won!")
        # exit the script
        break

    print()

    # ask the user to guess a character
    guess = input("Guess a character: ")

    # set the players guess to guesses
    guesses += guess

    # if the guess is not found in the secret word
    if guess not in word:
        # turns counter decreases with 1 (now 9)
        turns -= 1
        # print wrong
        print("Wrong")
        # how many turns are left
        print("You have", turns, "more guesses.")

        # if the turns are equal to zero
        if turns == 0:
            # print "You Lose"
            print("You Lose.")
```

## 创建一个魔法8球

魔法8球是我们将花一些时间研究和学习如何操作的最后一个项目。我们将制作一个程序，它的工作原理就像你小时候玩的魔法8球一样，但这次它将在你的计算机上。通过这个程序，我们可以向计算机提问，按下开始按钮，计算机将随机给出一个答案。

因为我们想在其中添加一些不同的元素，编写这个代码可能需要几分钟时间，但这并不会使过程变得更加困难。我们需要以下代码来制作我们自己的魔法8球：

```python
# Import the modules
import sys
import random

ans = True
while ans:
    question = input("Ask the magic eight ball a question: (press enter to quit)")
    answers = random.randint(1, 8)
    if question == "":
        sys.exit()
    elif answers == 1:
        print("It is certain")
    elif answers == 2:
        print("Outlook good")
    elif answers == 3:
        print("You may rely on it")
    elif answers == 4:
        print("Ask again later")
    elif answers == 5:
        print("Concentrate and ask again")
    elif answers == 6:
        print("Reply hazy, try again.")
    elif answers == 7:
        print("My reply is no")
    elif answers == 8:
        print("My sources say no")
```

我们为这个选择了八个选项，因为我们正在使用魔法8球。为了使其更有趣，你可以更改你想要的回复数量以及你想要更多还是更少。你在这里只使用 `elif` 语句，所以你可以不断添加它们以获得你想要的结果。
