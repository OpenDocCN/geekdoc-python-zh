### Python的历史

Python编程由Guido Van Rossum于1989年发现，当时他正在荷兰研究机构CWI进行一个项目，但后来该项目被终止。Guido成功地使用了一些基本语言，即所谓的ABC语言，来开发Python。根据Van Rossum的说法，Python语言的优势在于你可以保持其简单性，也可以将其扩展到更多平台，以同时支持多个平台。该设计允许系统轻松地与库和各种文件格式进行通信。

自推出以来，现在世界上许多程序员都在使用Python，事实上，许多工具被包含在内以提高操作效率。许多程序员采取了各种举措，向每个人普及如何使用Python编程语言，以及它如何帮助缓解对复杂计算机代码的恐惧。

然而，Python在几年前由Van Rossum开源，允许所有程序员访问甚至对其进行修改。这在编程领域改变了很多。例如，发布了Python 2.0。Python 2.0是面向社区的，使其在开发过程中透明。虽然许多人不使用Python，但仍有一些程序员和组织使用该版本的部分功能。

Python 3是一个独特的版本，于2008年发布。尽管该版本具有许多不同的功能，但它与前两个版本完全不同，并且更新程序并不容易。虽然此版本不向后兼容，但它有一个小型创建器来显示上传文件时需要更改的内容。

### Python中的通用术语

理解Python中使用的标准术语对你至关重要。它使你在开始时了解一切变得容易。以下是Python编程语言中最常见的术语：

- **函数：** 指当程序员使用调用程序时被调用的代码块。目标也是提供免费服务和准确计算。
- **类：** 用于开发用户定义对象的模板。它友好且易于每个人使用，包括初学者。
- **不可变：** 指具有固定值并包含在代码中的对象。这些可以是数字、字符串或元组。这样的对象无法更改。
- **文档字符串：** 指在函数、类定义和模块中显示的字符串。此对象始终在文档工具中可用。
- **列表：** 指Python内置的数据类型，包含排序的值。这些值包括字符串和数字。
- **IDLE：** 代表集成开发环境，允许用户在同一窗口中键入代码，同时解释和编辑它。最适合初学者，因为它是代码的绝佳示例。
- **交互式：** 由于其交互性质，Python已成为最适合初学者的编程语言。作为初学者，你可以在IDLE（解释器）中尝试许多事情，以查看它们的响应和效果。
- **三引号字符串：** 字符串帮助个人在字符串中使用单引号和双引号，从而轻松浏览不同的代码行。
- **对象：** 它指处于某种状态的所有数据，如属性、方法、定义的行为或值。
- **类型：** 指编程语言中的数据类别组，以及属性、函数和方法的差异。
- **元组：** 指Python内置的数据类型，是一个不变的值集合，尽管它包含一些可变的值。

### Python 语言的优势

使用 Python 程序比使用其他编程语言（如 C++ 和 Java）有许多优势。你会很高兴看到 Python 程序的可用性以及它学习和使用的简便性。理想情况下，这些是你现在可以使用的最佳编程语言，尤其是如果你是初学者。以下是使用 Python 语言的一些优势：

-   ❖ 易于使用、编写和阅读

许多程序员在使用 Java 和 C++ 等编程语言时面临一些挑战。由于它们的设计，它们很难查看。一个人必须花很多时间学习括号的使用，并且不容易识别这些编程语言中使用的一些单词。这些单词可能会吓到你，尤其是如果你刚开始接触编程语言。与 Java 和 C++ 语言不同，Python 不使用疯狂的括号。它只使用缩进，使页面易于阅读。它使用英语，使字符易于理解。

除了使用缩进，Python 还使用大量空格，使学习和阅读所需内容变得容易。它包含许多带有注释的地方，以便在程序让你困惑时理解或获得澄清。所以试试看，你会发现使用 Python 编程语言是多么容易。

-   ❖ 它以英语为主要语言

使用 Python 很容易，因为主要语言是英语。作为初学者，你将花更少的时间阅读和理解 Python 编程中使用的基本单词。因此，无论你是以英语为母语还是非母语，Python 都最适合你，因为大多数单词简单易懂。

Python 已经在一些计算机上可用

一些计算机（如 macOS 系统和 Ubuntu）预装了 Python。在这种情况下，你只需要下载文本解释器就可以开始 Python 编程。但是，如果你使用的是 Windows 计算机，则必须下载程序。事实上，即使你一开始没有安装 Python，它也能正常工作。

Python 与其他编程语言完美配合

第一次，你将单独使用 Python。但是，随着你继续编程，你会意识到 Python 可以与其他语言一起工作。你可以与 Python 一起使用的一些编程语言包括 C++ 和 JavaScript。尝试了解更多关于 Python 的知识以及它实际能做什么。随着时间的推移，你将能够发现许多事情。

Python 可用于测试许多东西

下载 Python 后，你需要下载测试解释器。测试解释器在使 Python 能够读取信息方面发挥着重要作用。最好使用 Windows 中可用的简单产品（如记事本）或其他解释器。

### 使用 Python 编程的缺点

虽然使用 Python 有许多优点，但必须认识到使用它的一些不利影响。由于 Python 的以下负面影响，一些人更喜欢使用其他编程语言（如 C++ 和 JavaScript）而不是 Python。

Python 速度慢

虽然 Python 与其他编程语言配合良好，并且适合初学者，但不幸的是，Python 对于寻求高速程序的程序员来说并不理想，因为它比其他选项翻译速度慢。速度水平取决于你翻译的内容，因为一些使用 Python 代码的基准测试比其他代码运行得更快。目前，世界各地的许多程序员正在尝试通过加快解释速度来解决这个问题。希望 Python 很快能以与 C 和 C++ 相同甚至更快的速度运行。

Python 在大多数移动浏览器中不可用

虽然 Python 对于拥有普通计算机的人来说运行良好，并且可以在许多服务器平台和桌面上访问，以帮助个人创建他们想要的代码，但它尚未准备好用于移动计算。程序员正在尝试将程序过渡到移动计算，以满足当今大量使用手机的人的需求。

-   ❖ 设计有限

### 2.1 - 变量

当你听到“变量”这个词时，你会想到什么？你很有可能会回想起代数课，并回忆起变量通常是一个代表某个数字的字母。换句话说，它是一个包含另一个项目表示的项目。通常，在 Python 和编程中，变量就是这样的。它们只是以易于访问和操作的形式存储的数据表示。让我们现在尝试创建一些变量。

```
X = 30
```

这是一个非常常见的变量用法。就像你在数学课上看到的，“X”代表数值 30。但是，变量也可以存储其他类型的数据，比如文本。

```
weather = "cloudy"
```

在这里，我们断言“weather”变量中存储了单词“cloudy”。当我们在 Python 中使用“等于”号时，与数学中不同，我们并不是说两个值相等。相反，我们是在“赋值”或在该变量中存储一个值。我们将在介绍运算符时详细讨论这一点，但目前，只需知道在 Python（以及通常在编程语言中）中，等号的使用方式与数学等号不同。

关于变量命名，只要名称遵守一些规则，变量基本上可以命名为任何名称：

-   它只包含字母、数字或下划线
-   **第一个字符不是数字**
-   **变量名不是保留关键字之一**

大写和小写字母都可以，任意数量的数字或下划线也可以，但变量名的第一个字符不能是数字。还有一些单词（如“input”、“print”和“while”）不能用作变量名，因为 Python 将这些单词保留用于特定命令。最后要记住的一件重要事情是变量名区分大小写，因此“varName”与“varname”是不同的变量。

虽然变量几乎可以命名为任何名称，但在实践中，创建变量名时你需要遵循一些准则。过去，有两种普遍接受的变量名格式化方式：驼峰式命名法和蛇形命名法。驼峰式命名法是指用大写字母连接复合词的做法（即 variableName、thisIsCamelCase）。同时，蛇形命名法只是在复合变量名之间添加下划线，如下所示：underscore_name、variable_name、this_is_snake_case。

根据 Python 风格指南，首选的变量风格是蛇形命名法。

除此之外，尝试给变量起有意义的名称。避免使用令人困惑的变量名（即 Cat = “dog”），而选择直观有意义的变量名（animal = “dog”）。

### 2.2 - 运算符

既然我们已经讨论了变量，让我们来谈谈运算符。运算符是执行功能的符号，用于以不同方式操作数据。

我们已经接触过一个运算符，即赋值运算符，由等号表示。如前所述，Python 中的“=”号与其在数学中的用法不同；在这里，它的意思是将运算符右侧的值赋给左侧的变量名。让我们尝试使用 = 运算符并为一些变量赋值。

### 2.3 - 组合运算符

赋值运算符可以与加法、减法和乘法等其他运算符组合使用。这使得我们能够一步完成操作，而无需分多步进行。为了更清楚地说明，这里有一个例子：

与其必须写：

```
R = R * 2
```

我们只需写：

```
R *= 2
```

尝试使用其他组合运算符，看看它们让值的更新变得多么简单。

### Python如何工作？

Python程序执行原理非常简单。我们都知道，用C/C++等编译语言编写的程序需要从源文件转换为计算机使用的机器语言，然后通过链接器链接后形成二进制可执行文件。运行程序时，可以从硬盘将二进制程序加载到内存中并运行。

然而，对于Python，Python源代码不需要编译成二进制代码。它可以直接从源代码运行程序。Python解释器将源代码转换为字节码，然后将编译后的字节码转发给Python虚拟机（PVM）执行。

当我们运行Python程序时，Python解释器执行两个步骤。

(1) 将源代码编译为字节码

编译后的字节码是Python特有的表达形式。它不是二进制机器代码，需要进一步编译才能被机器执行。这也是为什么Python代码无法像C/C++那样快速运行的原因。

如果Python进程在机器上具有写入权限，它会将程序的字节码保存为扩展名为.pyc的文件。如果Python无法在机器上写入字节码，则字节码将在内存中生成，并在程序结束时自动丢弃。构建程序时，最好授予Python在计算机上的写入权限，这样只要源代码不变，生成的.pyc文件就可以重复使用，以提高执行效率。

(2) 将编译后的字节码转发给Python虚拟机（PVM）执行。

PVM是Python Virtual Machine的缩写。它是Python的运行引擎，是Python系统的一部分。它是一个大型循环，迭代运行字节码指令，依次完成操作。

### 理解函数的概念

在这里花一两分钟，让你的思维活跃一下。思考一下，试着对函数到底是什么形成一些模糊的概念。

函数要么是用户定义的，要么是预定义的。无论哪种情况，它们的工作都是将代码组织在一个可调用的函数名中。Python 中有大量预定义的函数可用。我们已经反复使用了其中一些。

我们已经对内置和预定义的函数有了相当好的了解。这些包括但不限于 `input()`、`print()` 等等。然而，现在让我们看看如何创建我们自己的函数。

让我们从一个传统的方法开始，编写一段代码，用友好的问候欢迎用户。我们将把它存储为一个名为“welcome_message”的函数，以便稍后可以调用这个函数。

```
def welcome_message():
    print("Hello and welcome")
    print("Hope you have a great time")
```

```
print("Begin")
welcome_message()
print("End")
```

让我们开始学习，看看上面代码块中发生了什么。首先，为了创建我们的函数，我们需要先定义它。`def` 是一个关键字，Python 会看到它并立即明白你即将“定义”一个新函数。接下来，我们需要给函数命名。虽然你可以随意命名函数，但强烈建议并鼓励你使用易于理解且具有描述性的名称。如果我们把这个函数命名为 welcome_message 以外的任何名字，我们作为编写者可能知道它是什么，但对于其他任何程序员来说，他们可能无法理解。

每当你创建一个函数时，你都需要使用括号。你不必通过它传递任何信息，所以保持它们原样即可。现在，我们需要添加冒号。

当你在语句末尾使用冒号时会发生什么？你的光标会在下一行缩进。这意味着你的光标会稍微远离实际的起始点。这是为了向程序员表明，他/她即将输入的内容将对上面的命令或语句具有价值。在这种情况下，我们正在尝试定义函数。

然后让我们使用两个 print 命令并放置我们的问候消息。就是这样！你现在创建了你的第一个函数。你可以根据需要多次调用它。但是，如果你尝试在 `def` 命令之前一两行调用这个函数，Python 将不知道你在说什么。为什么？这完全与 Python 逐行读取程序的事实有关。当它到达你调用函数的那一行时，它会检查前面的行，但找不到任何相关的内容，因为实际的 `def` 步骤是在这之后执行的。

接下来，让我们使用我们的函数，看看它是如何工作的。记住，这个函数包含了两个可打印的消息给我们的用户。为了参考，我们现在创建一个“begin”和一个“end”消息。这将让我们和程序员知道常规消息在哪里，函数在哪里。如上所示，在两个 print 命令之间使用你的函数，括号留空。如果你愿意，你可以删除这些 print 命令，只输入你的函数名来查看结果。

给所有人一个小提示！如果你遇到烦人的波浪线，只需将鼠标悬停在上面，你就会发现预期或建议的解决方案是什么。在这种情况下，如果你删除两行空行，你应该会看到这样的建议：

每当你定义一个函数时，你总是需要在继续编写代码之前至少留出两个空行。

现在，让我们运行程序，你应该会看到所有的消息和我们的函数在运行。Python 启动了序列并首先读取了定义。这是 Python 仅为自己理解函数是什么的地方。当 Python 到达第六行，我们的 `print("Begin")` 消息开始时，实际的程序才被执行。在下一行，我们放置了我们的函数，这就是 Python 回忆起它刚刚学到的内容的地方。它快速执行了我们在函数内定义的一组代码并执行了相同的代码。最后，它在完成程序之前执行了最后一行。

这就是函数的创建和使用方式。现在，我们可以在同一个文件中根据需要多次使用这个函数。请注意，如果你打开一个新文件或处理一个未定义此函数的旧文件，你将无法使用这个新创建的函数。

在你未来的编程生涯中，当事情开始变得更困难时，记住创建你的函数并在适用的地方使用它们。它们将为你节省大量时间，并在某些地方提供帮助。这些函数用于需要不时执行某些操作或运算的情况。

### 使用各种函数

Python 的设计初衷是**简洁**。它旨在**最小化工作量，最大化输出**。如果你能明智地使用代码和函数，你一定能充分利用这门编程语言。值得注意的是，你所学到的关于 Python 及其函数、参数、方法等大部分知识，都将帮助你更快地学习其他语言，因此请务必留意。

### 参数

我们眼尖的读者可能已经注意到，几分钟前我们创建的函数有些特别。与大多数函数不同，我们根本没有在括号中传递任何信息。为什么会这样？一旦我们确切理解了 Python 中的**参数**是什么，我们就会明白。

参数用作接收信息的占位符。这些是我们（以及用户）提供给程序的信息，以便程序能更准确地工作。有些情况下，某些函数在执行基本操作时并不需要参数。然而，如果你为这些函数提供一个参数，它们将为你提供更具体的输出。当然，这取决于该参数是否可用。你不能强迫一个函数去做它设计之外的事情。

现在，让我们看看我们的函数。它确实缺少了什么。如果你现在打印 `welcome_user` 函数，它会显示所有内容，但完全不包含用户的名称。当然，如果我们能以某种方式使用这个函数来获取用户的名称并将其添加到问候语中，看起来会好得多。

幸运的是，我们确实可以这样做！为此，我们首先需要在开始定义函数的第一行添加 `name` 参数。只需在括号中输入 `name`，你就会看到文本变成灰色。这确认了该词已被添加为参数。现在，我们希望打印这个用户的名称以及我们在函数中定义的问候语。在这个例子中，让我们假设用户的名字是 Fred。

```
def welcome_message(name):
    print("Begin")
    print("Hello and welcome {name}!")
    print("Hope you have a great time")
    print("End")
```

```
welcome_message('Fred')
```

```
Begin
Hello and welcome Fred!
Hope you have a great time
End
```

终于！我们有了一个可以添加到问候语中的名字。你可以通过使用我们的函数并传递一个不同的名字来添加另一行代码。看看会发生什么。

当我们为一个函数设置了一个参数，然后在调用该函数时没有提供参数或括号中应包含的信息时，它会给我们一个错误，除了少数例外情况。

现在，让我们使我们的函数更具动态性，并添加另一个参数。让我们添加一个参数，允许程序打印用户的姓氏。现在，我们的代码应该看起来像这样：

```
def welcome_message(name, last_name):
    print("Hello and welcome {name} {last_name}!")
    print("Hope you have a great time")
    print("Begin")
    welcome_message('Fred', 'William')
    print("End")
```

这里要学习的要点，除了能够添加参数之外，是 `Fred` 和 `William` 是以特定顺序使用的。如果你输入顺序相反，Python 将按原样打印它们。这是因为它们相对于已定义参数的位置。Python 读取的第一个值，将自动将其与第一个参数关联。这可能会引起一些混淆，特别是当姓氏变成名字时。

这些参数被称为**位置参数**。为了进一步说明它们的重要性，让我们从上面的参数中移除姓氏。

```
print("Begin")
welcome_message('Fred')
print("End")
```

```
Traceback (most recent call last):
  File "C:/Users/Smith/PycharmProjects/MyFirstGo/PosArg.py", line 7, in <module>
    welcome_message('Fred')
TypeError: welcome_message() missing 1 required positional argument: 'last_name'
```

所以，系统不允许我们继续，因为我们移除了一个元素。这次，先输入姓氏，然后输入名字，看看是否有任何不同。现在运行程序时，你应该能看到：

```
print("Begin")
welcome_message('Fred', 'William')
print("End")
```

```
Begin
Hello and welcome William Fred!
Hope you have a great time
End
```

现在，顺序似乎起作用了。唯一的问题是它把名字搞错了。现在，姓氏被描绘并打印为名字。这相当尴尬，不是吗？

上述错误要么指出我们缺少一个必需的位置参数，要么显示我们把错误的名字放在了错误的位置。位置参数是那些位置非常重要的参数。如果你完全忽略了位置，最终会得到一个错误。如果你输入了其他内容，就像我们在最后一个例子中所做的那样，你会产生不正确的结果。要纠正它，只需在名字后面提供姓氏。

还有一种方法可以使用所谓的“关键字参数”来处理这些情况。这类参数的位置根本不重要，无论它们在括号中的位置如何，Python 都将继续正常工作。要传递关键字参数，你需要执行以下操作：

```
print("Begin")
welcome_message(last_name='William', name='Fred')
print("End")
```

```
Begin
Hello and welcome Fred William!
Hope you have a great time
End
```

现在看起来更像样了。事情正按照我们想要的方式发展。注意，即使我们写错了顺序，Python 也为我们挑选并排序了顺序。这是因为我们使用 `name=` 或 `last_name=` 参数将我们的条目或参数转换为关键字参数，并将其与参数结合。这允许 Python 提取信息并理解这两个参数中哪一个在顺序上排在前面，正如我们最初在函数中定义的那样。

事实上，你不会经常使用这些，但知道克服某些问题的方法总是一个优势。通常，你或其他程序员只要遵循规则，先输入名字再输入姓氏，就能轻松查看和读取数据。让代码尽可能简单，以便每个人都能阅读和理解。

> “那么，如果我使用数字而不是名字呢？”

这是一个很好的问题。这就是你需要使用关键字参数来表示这些值用途的地方。你可能正在运行一个涉及多个值的函数，这些值可能只有你自己能理解，但其他人将不知道它们来自哪里。你必须用相关的关键字参数标记每一个，以提高可读性。

### Return 语句

我们本可以在讨论 `if` 语句和其他内容时介绍什么是 `return` 语句。然而，在理解了参数和函数的概念之后再学习这个更有意义。

到目前为止，我们创建了一个允许我们通过使用参数来发送信息的函数。然而，当我们谈论 `return` 语句时，它们旨在进行某些计算并为我们提供结果，而不是我们向其提供值。

让我们通过创建第二个函数来更深入地了解这是如何工作的。这个函数的目的基于一个简单的数学技巧，我们大多数人可能在小时候都听说过或玩过。让一个用户想一个数字，然后让他们加减几个简单的数字。最终，你会给他们一个准确的结果，每个人都会感到震惊。现在，让我们揭示使用我们即将创建的这个函数会发生什么。

```
def magic_number(number):
    return number + 6 - 4 + 5 - number
```

这个简单的计算总是会返回值七。来吧，自己试试看。然而，为了让我们能够取回这些值，我们在这里使用了 `return` 语句。这告诉 Python 它应该为我们进行计算，然后只返回结果值，而不是单独打印每一个。

让我们测试运行一下我们的第二个函数：

```
result = magic_number(8329)
print(result)
```

看到结果现在显示为七了吗？你可以尝试将值更改为任何你喜欢的值，结果将继续保持为七。这只有在我们提供了 `return` 语句的情况下才可能实现。如果你去掉关键字 `return`，你最终会得到一个值为 `None` 的结果。程序仍然会运行，但结果将不再被计算或对我们有用。这是因为 Python 不会执行 `return` 阶段，因此不会按照我们希望的方式进行计算。

使用这些可以极大地增强你作为程序员或用户的体验。然而，在你投入并开始创建自己的函数之前，这里有一些预定义的函数可能会派上用场。创建一个函数却发现 Python 已经为你准备好了一个是毫无意义的。

### min() 和 max() - 当你遇到各种数值，并希望快速找出列表或数据集合中的最小值时，可以使用 `min()` 命令并输入这些数字。最小值将被打印出来。当然，`max()` 函数的作用正好相反！

### sum() - 这是一个非常实用的函数，可以让你快速将列表中的所有数字相加，并立即为你生成结果。对于浮点数的精度可能不尽如人意，但嘿，它能让你继续前进。

### type() - 你可能会遇到成行成行的代码，其中变量散布各处。你现在想找出这个变量最初是什么，以及它是什么类型的变量。使用 `type()` 函数，你可以快速了解你正在处理的是哪种类型的变量。它会返回诸如 'bool' 这样的值，表示该变量是布尔类型。
