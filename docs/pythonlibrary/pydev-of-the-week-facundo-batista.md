**你为什么开始使用 Python？**

14 年前，当我在 Movistar 工作时，我需要在服务器端处理一些日志。

服务器运行的是 SunOS(！).我知道 C 和其他语言不太适合做这项工作。我学习和使用 Perl 有几个月了，直到我发现 Python 并爱上了它。
