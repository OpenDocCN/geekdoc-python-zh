## 下载

### Docker

```
docker pull apachecn0/pythonlibrary-blog-zh
docker run -tid -p <port>:80 apachecn0/pythonlibrary-blog-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pythonlibrary-blog-zh
pythonlibrary-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```
