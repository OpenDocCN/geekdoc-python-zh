应用程序开发人员总是与文件打交道。每当您编写新的脚本或应用程序时，都会创建它们。你用微软 Word 写报告，保存电子邮件或下载书籍或音乐。文件到处都是。您的网络浏览器会下载大量小文件，让您的浏览体验更快。
