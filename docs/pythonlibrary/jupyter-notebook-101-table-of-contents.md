目录的内容或顺序可能会改变。不过，我会尝试以某种形式涵盖所有这些主题。如果有时间的话，我还会研究一些其他的主题，比如对笔记本进行单元测试。我的一些支持者还要求提供跨 Python 版本管理 Jupyter、使用 Conda 和如果你可以将笔记本用作程序的章节。我也会研究这些，以确定它们是否在本书的范围内，以及我是否有时间添加它们。
