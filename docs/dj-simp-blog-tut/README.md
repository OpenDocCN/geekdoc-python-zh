# Django 之歌：简易博客搭建教程

本书主要面向 `Django 学习者和博客开发者`, 其中的代码可以在作者的 github 上获得, 主要是在 Mac 上进行开发, 并没有在其他运行环境下做测试

希望达到的目标:

*   希望能写出一个系列文章, 我也不知道到底能写多少
*   能够让认真阅读这个系列的文章的人, 能在读完之后做出一个简单的博客
*   教会读者使用简单的 git 操作和 github
*   希望能够加深自己对`Django`的理解
