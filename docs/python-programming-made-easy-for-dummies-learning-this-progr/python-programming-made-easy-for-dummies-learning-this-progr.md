## Python

如今有许多编程语言被使用，数量之多以至于几乎不可能学会每一种。人们很容易认为程序员会满足于所有可用的语言来与计算机交流，但事实是，每年都有新的语言诞生。

开发一种新的编程语言并非出于一时兴起。程序员创造新语言都有充分的理由。每种编程语言都专注于特定的功能。此外，随着计算机硬件的发展，计算机软件以及代码和编程语言也随之演进。为了跟上这种持续的演变，程序员创造了一种新的语言，以简化硬件和软件之间的通信。例如，为了实现更精确设计的界面，一些程序员在认识到现有语言的局限性后，构建了一种独特的计算机语言。

Python在某些功能上表现出色，因此了解这些功能是什么非常重要。熟悉一门编程语言的一部分就是了解它的优势和劣势。

### 使用Python的理由

编程语言是由程序员为特定目的构建的。他们心中的需求有助于塑造语言，并决定语言的灵活性和实用性。Python的创建目标是打造一种语言，让程序员能够用它编写高效且实用的代码。

以下是您在编写应用程序时需要使用Python的原因：
- (a) 开发时间更短：与其他类似的编程语言（如Java和C/C++）相比，Python代码短5到10倍。这使得Python不那么复杂但更高效。这也意味着您在开发应用程序上花费更少的精力和时间，而有更多时间去探索它。
- (b) 学习时间更少：Python创造者的主要目标是创建一种规则更少、对初学者来说不那么难学的语言。这将帮助程序员创建更多可用的应用程序，而这些是使用更复杂和晦涩难懂的语言无法做到的。
- (c) 易于阅读：在检查错误或缺陷时，语言易于阅读和理解至关重要。如果语言过于复杂，编写和测试程序代码会很困难。Python代码比当今任何其他语言都更容易编写和阅读。这将使您更容易解释代码并进行必要的修改。

此外，以下是Python的推荐具体用途：
- (a) 编写基于浏览器的应用程序脚本
- (b) 创建更好的用户界面
- (c) 创建复杂的应用程序示例
- (d) 处理XML
- (e) 设计科学、数学和工程应用程序
- (f) 与数据库交互

#### Python与其他编程语言的比较

有时，将一门编程语言与其他语言进行比较可能非常主观。差异主要是个人偏好的问题，但有时这种比较可以通过可量化的科学数据来支持。

在阅读本节之前，必须注意世界上没有完美的、包罗万象的语言。您需要做的只是找到最适合特定功能或目标的语言。

##### (1) C#

有说法称微软的开发人员在创建C#时只是复制了Java。尽管如此，C#与Java相比仍有优缺点。根据微软的说法，他们创建C#的主要目标是创建一个更好的C/C++版本。当与C#相比时，Python享有以下优势：
- (a) 更好的工程和科学应用程序
- (b) 使用C/C++和Java时更具可扩展性
- (c) 更好的多平台支持
- (d) 更简洁（因此更小）的代码
- (e) 更容易学习和理解
- (f) 允许使用各种计算机环境。

##### (2) Java

与Python一样，Java一直是程序员的一站式解决方案。几十年来，程序员一直在寻找一种几乎可以在任何地方编写和运行的语言。Java旨在各种平台和计算机环境中表现出色。尽管有这种相似性，Python最重要的优势是：
- (a) 更简洁（因此更小）的代码
- (b) 更容易学习和理解
- (c) 更好的变量或数据容器，可以保存各种类型的数据，这些数据可以在运行不同程序时使用
- (d) 开发速度快数倍

##### (3) PERL

也称为实用提取和报告语言，PERL是一种擅长从不同数据库中获取和处理数据的编程语言。以下是Python更好的原因：
- (a) 更易于阅读和实现
- (b) 更好的Java集成
- (c) 更好的数据保护
- (d) 更容易学习和理解
- (e) 平台偏见更少

### 与Python交互

Python的构建旨在为用户提供多种方式来改善与计算机的交互。以集成开发环境（IDLE）为例。IDLE使Python用户更容易开发更好的应用程序。

然而，有时您确实需要仔细检查现有程序是否存在缺陷或错误。如果是这种情况，建议您使用Python的命令行版本。此版本比其他平台提供更好的用户界面，因为它需要更少的资源、更少的命令行开关，并且依赖于最低的系统要求来运行代码。

#### 启动Python

根据您在工作计算机上安装的平台，有多种方法可以打开Python的命令行。以下是您可以使用的方法列表：
- (a) 打开终端或命令提示符。只需输入Python，然后按回车键。程序员在需要更灵活地配置其Python环境时经常使用此功能。
- (b) 在文件管理器中找到Python文件夹，然后选择Python（命令行）选项。此选项让您使用默认设置配置的Python命令行版本。
- (c) 如果您使用的是Windows，您仍然需要安装Python。安装后，转到Python文件夹。通常位于此地址：C:Python33。双击Python.exe文件名。此方法打开一个默认设置配置的命令行。但是，使用此方法，您可以选择使用具有升级权限的凭据打开命令行。这允许系统运行使用受保护资源的程序。

在使用命令提示符运行Python时，您可以使用各种选项来提高其可用性。通过使用以下语法访问这些选项：

`python <options>`

将 `<options>` 替换为以下任何区分大小写的选项：
- `-a`：当正在运行的程序包含受限的Python功能时，此命令会在输出中包含红色标志或警告指示器。
- `-c cmd`：此选项在启动应用程序时使用从cmd获取的数据和信息。这还会告诉Python何时停止处理其他数据选项。
- `-d`：通过启动调试器选项来定位系统中的错误。
- `-h`：通过在屏幕上显示基本环境变量和有用的选项来启动程序。
- `-i`：这会打开一个Python接口，允许您检查正在运行的代码。
- `-m mod`：这允许Python将命令mod指定的库系统作为脚本的一部分运行。
- `-q`：这会阻止Python在交互式启动时打印敏感信息，如版本和版权信息。

输入命令 打开Python命令行版本后，预防性的第一步是检查您需要运行应用程序的选项是否有效。

命令是计算机用于执行特定任务或检查想法和数据的编码命令。与隐藏Python编程不那么花哨细节的IDLE不同，命令行界面为您提供第一手的经验和知识，关于## Python 的工作原理。

指令告诉计算机要做什么。它们是编码系统中的一系列步骤。这是通过输入 Python 熟悉的指令来实现的。作为回应，Python 将这些指令翻译成计算机可以理解的指令。例如，命令 `print()` 会在显示屏上显示该系统的结果。

为了让你快速了解命令的工作原理，请使用命令行界面执行以下步骤：

1. 使用本章前面阶段中列出的任何选项打开命令行版本。
2. 在命令行中，输入以下内容：
   `print("This is an example of how a command works.")`
3. 按回车键表示命令结束。然后命令行将显示：
   `This is an example of how a command works.`

##### 退出 Python

如果你想在玩了几个小时编程语言后关闭命令行，有两种标准方法可以做到这一点。还有其他关闭 Python 的方法，但建议你使用流行的方法，以确保下次使用时 Python 能正常运行。这些方法如下：
`quit()` 和 `exit()`
这些指令将关闭当前的 Python 会话。这些命令最棒的地方在于它们可以接受一个可选参数。例如，你可以使用 `quit(5)` 或 `exit(5)` 来关闭 Python，同时 `ERRORLEVEL` 变量达到数值五。

为了说明如何优雅地退出 Python，请执行以下步骤：

1. 打开终端或命令提示符。
2. 输入 `python` 然后按回车键。
3. 输入 `quit()`。

###### IDLE 的使用

虽然你可以使用任何文本编辑器来编写 Python 程序，但 IDLE 提供了一个更高效的平台来编写 Python 代码。交互式开发环境是经验丰富的程序员最广泛使用的平台。IDLE 可以执行以下与 Python 相关的任务：(a) 编写 Python 代码，(b) 保存和运行 Python 数据和文件，(c) 执行简单的编辑，如复制、剪切和粘贴，(d) 浏览和发现特定的 Python 文档，(e) 通过执行调试策略来消除错误，以及 (f) 突出显示和理解来自独特文本数据的重要代码和关键字。
上述大多数功能都可以使用命令行界面执行，但这需要时间和精力。例如，你可以使用命令行界面执行调试方法，但这是以手动且容易出错的方式执行的。这使得使用 IDLE 的难度降低了好几倍。

## 第 4 章。Python 数据类型

为了有效地处理事实，Python 将输入信息分类为多个类别。这些类别作为信息存储区域，可以在 Python 平台运行时随时访问。此外，这还在修改特定事实位时为准则提供了形状。否则，系统可能无法工作。

### 变量

与代数类似，变量包含分配给它们的特定值。在编写更复杂的程序代码时，将记录保存在变量中至关重要。
在 Python 中，变量是一种存储容器，每次你想使用或处理其中保存的记录时都可以访问它。例如，如果你需要用户输入他们的姓氏，你可以创建一个变量来存储与他们姓氏相对应的输入。
为了让他们更容易完成工作，程序员使用变量来确保从最终用户收集的统计数据是安全的。任何形式的事实，从日期和时间、数字和字母，都可以保存在变量中。
为了将统计数据放入变量中，你需要创建一个使用赋值运算符 `=` 的命令。例如，如果你想将数字 3 分配给变量 `defVar`，请输入 `defVar = 3` 然后按回车键。

### 数字类型

如果在算术中，数字 5 和 5.0 是相同的，但 Python 对它们的看法不同。为了使 Python 能够处理这种多样性，它知道 5 和 5.0 经历不同的记录处理策略。以下是你应该熟悉的数字类型：

1. **整数**
   整数是任何完整的数字。例如，5 是一个完整的数字，但由于小数部分，5.0 不是一个完整的数字。在这种情况下，Python 知道 5 是一个整数，因此可以通过 `int` 数据类型来表征。
   与变量类似，数字类型，特别是整数，在容量方面有限制。如果候选者尝试处理超出 Python 正常限制的值，将会发生错误。

在最大结构中，整数可以处理-9,223,372,036,854,775,808到其有效对应值之间的任何数字。这个数量是64位整数的最大值。
`int`变量具有有趣的功能，这些功能因基数类型而异，例如：(a) 仅使用1和0的二进制（Base 2），(b) 使用0到7数字的八进制（Base 8），(c) 类似于十进制系统的十进制（Base 10），(d) 使用0到9数字和字母A到F作为数字的十六进制（Base 16）。
当告诉Python应用特殊的基数系统时，你只需上传0然后是一个特殊的对应字母到范围。以下是Python用来表示基数系统转换的字母：
B – 二进制（Base 2）
O – 八进制（Base 8）
X – 十六进制（Base 16）
例如，`0b1100`是二进制，`0o741`是八进制，`0x123aa`是十六进制。此外，要将它们的值转换为其他基数，你可以使用以下指令：
`bin()` – 二进制
`oct()` – 八进制
`hex()` – 十六进制

#### 2. 浮点值

Python将任何带有小数点的数字（即使只是`.0`）视为浮点值。如前面的例子所示，`5.0`不是整数；相反，它是一个浮点数。没有必要担心，因为一旦一个数字有小数部分，它就是一个浮点数。这也意味着这种数据存储在`float`数据类型中。浮点值相对于整数的主要优势是其更大的存储空间，允许存储非常大或非常小的值。这并不意味着它有无限的存储空间。实际上，它可以包含高达1.7976931348623157 × 10^308和低至± 2.2250738585072014 × 10^-308的值。

有几种方法可以使用浮点值赋值。第一种是直接赋值，第二种是使用上面提供的科学记数法。理解负指数提供等效的分数。

#### 3. 复数

也许从基础数学中回顾一下，复数是由一个虚数和一个实数配对组成的数字。这通常用于以下领域：(a) 动态系统，(b) 计算机图形学，(c) 量子力学，(d) 流体动力学和(e) 电气工程。

复数还有许多其他用途。由于Python是少数能够处理复数的编程语言之一，这些领域的人们熟悉这种编程语言。

### 布尔值

Python有一种方法可以产生一个直接的答案，这个答案可以是真或假。事实上，计算机可以被指示遵循一个称为布尔代数的数学分支。尽管在乔治·布尔的时代（布尔代数由此得名）还没有计算机，但如今许多计算机语言在编写算法以做出决策或响应时都依赖于所述数学。事实上，布尔代数自1854年以来就存在，远在第一台现代计算机出现之前。
如果程序员需要布尔代数，他们参考`bool`数据类型。变量只能包含两个响应值：真或假。这种数据类型允许程序员使用上述响应赋值。例如：
`MyBool = 15 < 3.2`
将给出一个由值`False`表示的响应，因为15不小于3.2。

### 字符串

这种数据类型类似于代表确切含义的单词，使人类更容易在日常生活中使用它。字符串是一组字符（不一定是可理解的单词），由双引号连接在一起。例如：
`ThisString = "I love to learn the Python language!"`
此命令将短语"I love to learn the Python language!"赋值给变量`thisString`。为了让你理解计算机如何理解这个命令，你应该知道计算机看不到字母。相反，双引号内的每个字母都由一个唯一的数字表示。例如，字母A由65表示，因此当你输入命令：`ord("A")`时，该命令将输出65。你可以尝试其他字母来检查它们的数字等效值。由于计算机无法处理字母，你必须将字符串转换为数字。在这种情况下，你可以使用命令`float()`或`int()`。例如：`ThisInt = int("6766")`将整数6766而不是字符串6766赋值给变量`thisInt`。Python还允许用户使用命令`str()`将数字转换为字符串等效值。例如：`ThisString = str(67.6766)`创建一个包含值67.6766的字符串，然后将其赋值给变量`thisString`。这允许程序员在将值存储在变量中之前，在将字母转换为数字或反之亦然之间来回转换。

### 日期和时间

几乎每个应用程序都需要日期和时间才能有效工作。它安排任务应在哪个特定日期和时间完成。由于计算机只理解数字，日期和时间的概念在计算机方面不存在。它们能够与日期和时间输入交互的唯一方法是将它们转换为可读的值，这些值可以存储在变量中。命令`import datetime`允许程序员访问日期和时间。为了从计算机内部的时钟获取当前时间，你可以使用命令`datetime.datetime.now()`。例如，如果日期是2015年6月12日，时间是凌晨4:00，你输入：`datetime.datetime.now()`，该命令将响应以下输出：`datetime.datetime(2015, 6, 12, 4, 0, 0, 0)`。由于此输出可能难以阅读和解释，你可以使用：`str(datetime.datetime.now().date())`来安排输出。Python的优点之一是该语言有一个`time()`命令，可用于获取确切时间。以下保留字也可用于评估其相应的精确值：`microsecond`、`second`、`minute`、`hour`、`year`、`month`和`day`。

### 执行重复任务

本书提供了关于如何执行单个循环任务的简洁示例。然而，有时应用程序可能需要持续和连续的评估，直到达到某个点，达到期望的值并因此进行评估。Python允许使用`for`语句执行重复任务。

#### For语句

对于Python用户来说，`for`语句是初学者需要学习的第一个命令。几乎所有已知的传统计算机语言都使用`for`语句。`for`语句确定程序中循环将执行多少次。与后续的循环命令不同，`for`循环中的重复次数已经确定。

这是一个`for`语句的例子：`for Alpha in "Good Morning, Python!":`
该命令由关键字`for`引入。下一个单词是将包含赋值的变量。在示例中，变量的名称是`Alpha`。单词`in`表示字符串"Good Morning, Python!"的开始。冒号结束整个命令。

下一行必须缩进，以表示`for`语句下所有缩进的命令将作为计算机将遵循的指令循环，作为`for`循环的一部分。

## 软件，你可以因此创建一个。实际上，编写自己的函数可以让你节省更多时间。甚至可以将这些用户定义的函数应用到你的其他应用程序中。只是不要忘记使用函数时附带的规则。

首先，你必须以 `def` 开始你的代码块。然后，用括号和函数调用跟在后面。确保将你的输入参数或实参放在括号内。如果需要，你甚至可以使用这些括号来定义参数。在代码块的开头，不要忘记包含一个冒号（:）并使用缩进。

函数中的第一个语句是可选的。所以，如果你不需要包含一个，这完全没问题。尽管如此，你总是可以将其用作文档字符串或函数的 docstring。如果你想退出一个函数，你必须使用 `return` 语句。你也可以使用 `return` 将表达式传递回其调用者。但是，如果没有参数而你尝试使用 `return`，它可能是多余的，类似于 `return None`。

在编写带有函数的程序时，记得包含函数名，这实际上是你的函数的调用。在其下方，你必须放置你的函数的代码。确保它是缩进的；因为如果不是，它将不会被读取为一个程序。在 Python 中，函数在程序中被视为独立的。你的计算机不会理解一个函数，但它会理解它返回的值。

这是什么意思？例如，你需要将值 7 存储在变量 `d` 中。当你运行你的程序时，你的计算机不会识别变量 `d`。相反，它看到的只是你想要存储在 `d` 中的值 7。一个函数在运行时表现为它被赋予的任何值。它实际上是一个你提供参数的小程序。它运行自身然后返回一个值。

默认情况下，参数具有位置行为。它必须以与定义时相同的方式被调用。看看这个例子：

```python
def functionname ( parameters ):
    "function_docstring"
    function_suite
    return [ expression ]
```

```python
def printme ( str ):
    "the example prints out a passed string into a function."
    print str
    return
```

在你可以有效地定义一个函数之前调用它，你需要提供名称、形式和参数规范。提供函数调用和代码结构至关重要。此外，你需要提供参数定义。如果你已经确定了函数的基本形式，那么你可以从另一个函数调用它来执行它。但是，如果你希望直接从提示符调用它，你也可以这样做。

```python
    return;
```

### 按引用传递与按值传递

在 Python 中，引用传递所有参数和实参。这意味着函数中参数所引用的任何内容都指向你对其所做的任何修改。例如：

```python
# this is where you put the function definition
def changeme ( mylist ):
    "It changes a passed list within a function"
    mylist.append ( [ 5, 6, 7, 8 ] );
    print "The values inside the function are: ", mylist
    return

# you can call the changeme function here
mylist = [ 20, 30, 40 ];
changeme ( mylist );
print "The values outside of the function are: ", mylist
```

当你运行上面显示的代码时，你将获得以下输出：

```
The values inside the function are: [ 20, 30, 40, [ 5, 6, 7, 8 ] ]
The values outside of the function are: [ 20, 30, 40, [ 5, 6, 7, 8 ] ]
```

关于通过引用传递的参数，看看这个：

```python
# this is where you put a function definition
def changeme ( mylist ):
    "It changes a passed list in the function"
    mylist = [ 5, 6, 7, 8 ];
    print "The values in the function are: ", mylist
    return

# you can call the changeme function here
mylist = [ 20, 30, 40 ];
changeme ( mylist );
print "The values outside of the function are: ", mylist
```

参数 `mylist` 对函数 `changeme` 是局部的。当你在函数内部更改它时，你将看不到任何变化。该函数对其没有任何影响，你将获得以下输出：

```
The values in the function are: [ 5, 6, 7, 8]
The values outside of the function are [ 20, 30, 40 ]
```

如你所见，引用在被调用的函数中已被覆盖。

### 匿名函数
在 Python 中，支持匿名函数。它们可以在运行时使用 `lambda` 构造创建。这些函数不绑定到任何名称。请注意，这里的 `lambda` 与函数式编程语言中使用的 `lambda` 不同。基本上，Python 的 `lambda` 旨在成为创建函数的工具，通常称为函数对象。然而，它仍然功能强大，并经常与 `filter ( )`、`reduce ( )`、`map ( )` 和其他函数式概念结合使用。

匿名函数不使用 `def` 声明，而是使用 `lambda`。`Def` 和 `lambda` 实际上是 Python 构建函数的主要工具。请记住，有一些规则支配着匿名函数。首先，`lambda` 类型可以包含任意数量的参数。但是，它们应该只返回一个表达式作为值。不可能使用多个表达式或命令。此外，匿名函数不允许直接调用打印，因为 `lambda` 需要命名某些表达式。

一些用户认为 `lambda` 是一个函数，尽管是单行版本。这是一个常见的误解。`lambda` 在传递函数堆栈分配以进行调用方面，实际上并不像 C 和 C++ 中的内联语句。请注意，`lambda` 有自己的命名空间。因此，它不允许访问变量，除非是全局命名空间中的变量和参数列表中的变量。`lambda` 的语法如下：
```python
lambda [ arg1 [ ,arg2 , . . . argn ] ] : Expression
```

### 列表

列表是灵活性最高的复合数据类型。它包含括在方括号（`[ ]`）中、并以特定顺序用逗号（`,`）分隔的对象。它实现了序列操作，并帮助你从中删除和添加元素。

以下是列表的一个典型示例：

```python
#!/usr/bin/env python
list = [ 'wxyz', 3.14159265359, 'wendydawn', 'david' ]
shortlist = [123]
print(list)  # This prints the whole listing.
print(list[0])  # This prints the primary element inside the listing.
print(list[3:14])  # This prints the fourth to the fourteenth factors inside the listing.
print(list[2:])  # This prints the third element onwards.
print(shortlist * 3)  # This prints the list three instances.
print(list + shortlist)  # This prints the concatenated lists.
```

运行上述代码会得到以下输出：

```
[ 'wxyz', 3.1415926535900001, 'wendydawn', 'david' ]
wxyz
[ 'david' ]
[ 'wendydawn', 'david' ]
[ 123, 123, 123]
[ 'wxyz', 3.1415926535900001, 'wendydawn', 'david', 123 ]
```

#### 创建列表

为了创建一个列表，你需要将表达式放在方括号内，如下所示：

```python
W = [ ]
W = [ expression, . . . ]
```

在 Python 中，也支持称为列表推导式的计算列表。其语法如下：

```python
W = [ expression for a variable in a sequence ]
```

其中，表达式会对序列中的每个项目求值一次。这个表达式可以是任何东西。它可以包含任何形式的项目，甚至其他列表。它也可以包含对特定项目的多个引用。

用户也可以使用 Python 内置的 `list` 函数来创建列表。例如：

```python
W = list()  # This list is empty.
W = list(collection)
W = list(expression for a variable in a series)
```

这可以是任何类型的可迭代对象或集合项，包括生成器和元组。如果你传入另一个列表，`list` 函数会创建一个副本。请记住，每次执行 `[ ]` 表达式时都会创建一个新列表。当你将一个列表赋值给一个变量时，你永远无法创建一个新列表。请看以下示例：

```python
W = D = [ ]  # the two names point to the same list.
W = [ ]
D = W  # the two names factor to the same listing.
W = [ ]; D = [ ]  # those lists are impartial.
```

#### 访问列表

列表实现了通用的序列接口。它对 `*` 和 `+` 运算符的响应与字符串类似，意味着它们也能进行连接和重复操作，只不过它们产生的是一个新列表而不是字符串。列表也明确响应所有用于字符串的通用序列操作。

例如，`len(L)` 返回列表中有多少个项目，`L[i:j]` 返回一个包含 i 和 j 之间所有项目的新列表，`L[i]` 返回索引为 0 的第一个项目或指定索引处的项目。当你传入一个负索引时，列表的长度会被加到索引上。你可以使用 `L[-1]` 来访问列表的最后一个项目。

如果结果索引超出列表范围，会引发 `IndexError` 异常。切片被视为边界，结果包含它们之间的所有对象。列表也支持切片步长。以下是一些切片步长的示例：

```python
seq = L[start:stop:step]
seq = L[1::2]  # It receives every other object, beginning with the second one
seq = L[::2]  # It receives each different item, starting with the first
```

为了帮助你更好地理解列表，这里提供更多示例：

**输出：**


#### 编辑列表

在 Python 中，可以向现有序列添加对象。例如，`append` 将一个项目添加到列表末尾，而 `extend` 将另一个序列或列表中的项目添加到末尾。`insert` 在指定索引处插入一个对象，并将其他项目向右移动。也可以删除项目。例如，你可以使用 `del` 来删除切片标识的一个或所有项目。你可以使用 `pop` 来删除一个项目并返回它。你还可以使用 `remove` 来查找一个特定项目，然后从列表中删除第一个匹配的对象。`del` 和 `pop` 非常相似，只是 `pop` 可以返回被删除的项目。

你可以对列表做的另一件事是反转它们的顺序。如果你想这样做，可以使用 `L.reverse()`。反转列表的顺序实际上非常简单且快速。事实上，当你需要在列表开头插入和删除某些项目时，临时反转列表可以加快速度。

请记住，`for – in` 语句会保持内部索引。它在循环的每次迭代中都会递增。这意味着如果你决定## 调整正在循环遍历的列表

如果你在循环遍历列表时对其进行调整，索引可能会不同步。你甚至可能跳过某些元素或重复处理相同的元素。这显然不是一个理想的结果。要解决此类问题，你可以循环遍历列表的副本。例如：

```python
for object in L[:]:
    if not condition:

```

```python
out = []

```

一种常见做法是对列表中的每个元素应用一个函数，然后用函数的返回值更新列表。例如：

```python
for index, item in enumerate(L):

for object in L:

```

如果你想重写上面的例子并使其更简单，可以使用 `map`。这是一个内置函数，能使你的代码更高效。这是因为你的函数项只被获取一次。此外，你可以使用 `map` 将上面的例子重写如下：

```python
out = [function(object) for object in L]
```


```python
def examine(x, y):

L.sort(evaluate)
```

### 元组

元组本质上是不可变对象的序列，与列表非常相似。它们的主要目的是防止事物发生变化并保持其稳定。然而，与列表不同，它们不允许被修改，并且包含圆括号而不是方括号。你无法在元组中添加、删除或更改元素。

但是，如果你犯了一个错误，更改了一个不应该更改的变量怎么办？好吧，你可以通过将元组转换为列表或反之亦然来修复此错误。此外，与列表不同，元组不是同构序列。它们是异构数据结构，因此它们的实体具有不同的含义。请记住，元组有结构，而列表有顺序。

如何创建元组？很简单：只需用逗号分隔值或将它们放在圆括号内。例如：

```python
tup1 = ('algebra', 'physics', 1900, 2015)
tup2 = (5, 6, 7, 8)
tup3 = "w", "x", "y", "z"
```

也可以创建一个不包含任何值的空元组：

```python
tup1 = ()
```

请注意，即使元组只有一个值，你也需要放置一个逗号。例如：

```python
tup1 = (28,)
```

元组索引从零开始，并且可以连接和切片。

#### 切片元组

你可以通过切片元组来引用多个值。例如：

```python
tup[1:5]
```

#### 元组赋值

赋值左侧的变量元组可以被赋值为赋值右侧元组的值。请看以下示例：

```python
(name, middle_name, surname, ID_number, profession) = Chuck Norris
```

这一行执行了相当于五个赋值语句的操作。但是，如果你的元组中的元素数量与左侧使用的变量数量不匹配，你的程序将无法运行。

你可以将元组赋值视为元组打包和/或元组解包。例如：

```python
w = ("Wendy", 99, "WD")  # 这是元组打包的一个例子
```

关于元组解包，你将值解包到变量或名称中。例如：

```python
w = ("Wendy", 99, "WD")
(call, degree, initials) = w  # 这是元组解包的一个例子
```

有时，交换值是理想的。使用普通的赋值语句，你需要使用一个临时变量。因此，如果你想交换 x 和 y，如下例所示，你需要使用元组赋值：

```python
temp = x
x = y
y = temp
```

为了解决这个问题，你可以写：

```python
(x, y) = (y, x)
```

在左侧，你可以看到一个变量元组。在右侧，你可以看到如果你这样使用，将会引发 `ValueError`，因为你希望解包的值超过了3个。

#### 元组作为返回值

请记住，一个函数最有效地方式是返回一个值。但是，如果你将值转换为元组，就可以返回多个值。很多时候，用户希望了解一年的确切日期、最低和最高排名，或者标准差。计算给定半径的圆的周长和面积也是一个常见的数学问题。因此，利用这种情况，你可以这样编写：

```python
def f(r):
    """Return (area, circumference) of a circle with radius r"""
    a = math.pi * r * r
    c = 2 * math.pi * r
    return (a, c)
```
