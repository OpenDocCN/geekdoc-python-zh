http://pypi.python.org/pypi/pudb

模块 pdb 为 Python 程序定义了一个交互式源代码调试器。
http://docs.python.org/2/library/pdb.html

**皮林特**
分析 Python 源代码，寻找 bug 和质量差的迹象。
