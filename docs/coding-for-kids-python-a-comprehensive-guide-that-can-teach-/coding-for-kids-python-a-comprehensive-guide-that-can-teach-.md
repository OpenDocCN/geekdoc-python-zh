### 2.1 Python 编程中的变量

在 Python 中，标识符被称为变量。机器或计算机的一个内存区域被称为变量。Python 不需要你选择这些类型的标识符，因为它是一种功能全面的编程语言，能够自行对其变量进行分类。

此外，在 Python 编程中，变量或标识符是具有不同数据类型的内存位置，例如整数或字符。Python 中的变量是可操作和可更改的，因为它们通常与一组操作结合使用。建议使用小写字母 [例如 a, b, c, d,..., z] 作为变量名。

#### 变量或标识符的命名

因子是标识符/变量的位置。你的 Python 应用程序中使用的字面系数和整数是使用变量创建的。以下是 Python 变量命名约定。

- 变量或标识符的名称必须是单个字母或下划线 "_"。
- 除了基本字母外，其余字符可以是字母（小写 "a-z"、大写 "A-Z"、下划线或数字 "0-9"）。
- 标识符/变量的名称不得包含任何空白/空区域，以及任何特殊或异常字符，如 "! @, #, 百分号, 和 *"。
- 变量的名称不能是 Python 语法中的关键字。
- 变量区分大小写。例如，术语 alien 和 Alien 不可互换。
- 有效的标识符/变量包括 r203, _m, f90 等。
- 无效的标识符/变量包括 90, m percent 809, f90x69 等。

#### 多重赋值

- 多重赋值是指 Python 将单个位置分配给多个标识符/变量时使用的术语。它可以通过两种方式使用：同时为多个标识符/变量定义单个位置，或同时为多个标识符/变量声明多个位置。
- 在不同时间，将相同的属性或数量分配给大量变量/标识符。

#### 在同一变量上赋值的示例

打开 Python 控制台或 IDE，编写声明变量的命令。

```

```

### 4.1 什么是列表？

列表类似于其他语言（例如 C++ 中的 `vector` 和 Java 中的 `ArrayList`）中声明的动态数组。列表不必始终是同构的，这使其成为 Python 中一个强大的工具。整数、字符串和对象等数据类型都可以在同一个列表中找到。列表是可变的，这意味着它们在创建后可以被修改。

在 Python 中，列表是有序的，并且有计数。列表的元素按特定顺序索引，其中 0 作为第一个索引。列表中的每个元素在列表中都有其独立的位置，这允许元素在列表中重复，同时仍然拥有各自不同的位置。

我们使用列表的方式与在 C 或 C++ 中使用数组的方式相同。然而，列表可以包含各种类型的数据。列表中存储的项目通常用逗号 `,` 分隔，并用方括号 `[ ]` 括起来。

可以使用切片运算符 `[:]` 来访问列表的元素。加法运算符 `+` 和乘法运算符 `*` 对列表的作用方式与对字符串的作用方式相同。

#### 列表的特性

- 列表是按字母顺序排列的。
- 可以通过索引访问列表的元素。
- 列表代表可变类型。
- 列表的类型是可变的。

### 4.2 在 Python 中创建列表

在 Python 编程中，通过将所有项目（元素）放在方括号 `[]` 内并用逗号分隔来生成列表。

它可以包含任意数量的不同类别（整数、浮点数、字符串等）的元素。

```

```

列表也可以将另一个列表作为一项。这称为嵌套列表。

```
#### 嵌套列表

```

#### 列表的索引

要访问列表中的特定项目，我们可以使用索引运算符 `[]`。在 Python 中，索引从 0 开始并依次递增。因此，一个包含五个成员的列表的索引范围将从 0 到 4。
如果你尝试访问此处未列出的索引，你将得到一个 `IndexError`。索引必须使用整数。我们不能使用浮点数或其他类型，因为会发生 `TypeError`。
列表中有两种类型的索引：

1.  **正向索引** -- 在这种情况下，索引从 0 开始，从左向右移动。
    示例：

    ```
    # 列表索引

    my_list = ['p', 'r', 'o', 'b', 'e']
    # 输出: p
    print(my_list[0])

    # 输出: o
    print(my_list[2])

    # 输出: e
    print(my_list[4])

    # 嵌套列表
    n_list = ["Happy", [2, 0, 1, 5]]

    # 嵌套索引
    print(n_list[0][1])

    print(n_list[1][3])

    # 错误！索引只能使用整数
    print(my_list[4.0])
    ```

2.  **负向索引** — 这种方法从右向左索引，最右边的元素具有 -1 的索引值。Python 序列支持负向索引。最后一项由索引 -1 表示，倒数第二项由索引 -2 表示，依此类推。
    示例：

    ```
    # 列表中的负向索引
    my_list = ['p','r','o','b','e']

    print(my_list[-1])

    print(my_list[-5])
    ```

    运行上述程序后，我们将得到以下输出：

    ```
    e
    p
    ```

#### 创建 Python 字典

在 Python 中，通过将一系列条目括在花括号中并用逗号分隔来构建字典。字典存储一对值，其中一个是键，另一个是 **键：值（ ）** 对元素。在字典中，值可以是任何数据类型，并且可以重复，但键不能被复制，必须是不可变的。

字典键区分大小写。因此，两个名称相同但大小写不同的键将被解释为不同。

#### 示例：

```

#### 使用整数键

print(Dict)

#### 创建一个字典
#### 使用混合键
Dict = {'Name': 'Geeks', 1: [1, 2, 3, 4]}
print("\n使用混合键的字典: ")
print(Dict)
```

### 7.2 变量的作用域

全局变量是在文件主体中定义的变量。它在整个文件中都可见，也可以在任何导入它的文件中可见。由于其广泛的影响，全局变量可能产生不可预见的后果，这就是为什么我们几乎不应该使用它们。全局命名空间应仅包含旨在全局使用的内容，例如函数和类。

在函数内部定义的变量被认为是该函数的局部变量。它从函数开始到结束都可用，并且只要函数在运行，它就存在。函数规范中的参数名称充当局部变量，但它们包含我们在调用函数时传入的值。当我们在函数内部使用赋值运算符 (=) 时，除非局部作用域中已存在同名变量，否则它默认创建一个新的局部变量。

以下是不同作用域中变量的示例：

```

if a == 0:
    # This is still a global variable
    b = 1

def my_function(c):
    # this is a local variable
    d = 3
    print(c)
    print(d)
```

## 第8章：Python中的模块和包

面向对象Python编程最重要的组成部分是模块和包。在Python中，我们经常使用它们来更好地理解如何以逻辑方式使用代码。这些编程方法也存在于其他高级计算机编程语言中，并且它们附带了许多著名的框架。通过利用它们，可以将编程的复杂性简化为简单的编码。让我们分别看看每一个。

模块化编程是将一个大型、复杂的编程任务分解为更小、更易于管理的子任务或**模块**或**包**的技术。然后，各个模块可以像拼图一样组合在一起，形成一个更大的应用程序。

**在大型应用程序中模块化代码有多种优势**

**简单性：** 模块通常专注于问题的一个小部分，而不是整个问题。如果你正在处理单个模块，你将有一个更小的问题领域需要理解。这有助于开发并减少错误的可能性。

**可维护性：** 模块通常用于在不同的问题领域之间建立逻辑边界。如果模块的编写方式减少了相互依赖性，那么对单个模块的修改不太可能影响程序的其他部分。（你甚至可能能够在不了解应用程序其余部分的情况下对模块进行修改。）这使得一大群程序员在项目上协作变得更加可行。

**可重用性：** 在单个模块中创建的功能可以很容易地被程序的其他部分重用（通过适当指定的接口）。不再需要复制代码。

**作用域：** 模块通常有自己的命名空间，这有助于避免程序不同部分中的标识符冲突。（Python的禅宗原则之一是命名空间是一个绝妙的主意！）
