> 年轻的弗兰肯斯坦，1974 年

写一本书是一项巨大的努力。

我不希望这听起来像是对我忘记修复或引入的许多错误的一个借口。我只想说明，你所阅读的内容是我花费了无数个小时思考如何解释某个概念、与自己争论段落是否应该放在另一个段落之前，以及不断尝试将我在项目上写的代码和 Git 提交与书中的内容相协调的结果。既然你读到这一点，我想真诚地感谢你参与这次冒险。

我并没有遵循我收到的所有建议，但我仔细阅读了人们给我发送的所有问题和信息。很抱歉我没有能够把我想写的所有内容或你建议的所有内容都放入书中。我与编辑没有承诺，但某个时候，即使是自出版的书也必须被称为完成。我希望会有更多的版本，所以我只是推迟了这项工作。在此期间，你为什么不写一篇关于清洁架构的博客文章呢？

感谢阅读我的书！
