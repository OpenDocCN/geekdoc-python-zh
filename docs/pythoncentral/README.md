### Docker

```
docker pull apachecn0/pythoncentral-blog-zh

# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pythoncentral-blog-zh
pythoncentral-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```
