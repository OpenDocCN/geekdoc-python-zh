## 第七章：Python——面向对象编程

Python的另一个独特之处是它是一个面向对象（OOP）的程序。面向对象编程是基于数据而非逻辑，或基于对象而非动作的编程。这个概念被应用于大多数编程语言；C语言是唯一的例外。这个概念围绕着继承、多态、抽象、封装等目的展开。理解面向对象的基本概念非常重要，特别是对于Python、C++以及Java用户。这就是将本章包含在本书中的主要原因。从基础开始，我们将逐步深入到高级概念。
